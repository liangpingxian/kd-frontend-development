# Kingscript Docs

这个目录存放“规则、流程、开发范式和专题指南”类共享文档，适合回答：

- 某类开发能力适不适合当前场景
- 某种实现方式有哪些配置规则和边界
- 某个专题应该先看哪份开发指南

## 使用建议

- 先按主题进入最接近的子目录，再打开对应 `README.md`
- 如果问题偏“怎么写代码”，优先回到 `references/examples/`
- 如果问题偏“API 是什么、方法能不能调”，优先回到 `references/sdk/`
- 如果问题偏“语法、命名、模块限制”，优先回到 `references/language/`
