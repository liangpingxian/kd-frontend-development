## 2、事件方法

### 按模板导出时，删除每个采购申请单中金额小于500的分录行
```kingscript
/**
 * 系统按条件查询出待导出的数据后，即将执行导出流程时，触发当前事件
 * 用户可修改查询完毕的数据，如移除部分数据不作导出，或按自定义规则修改部分数据
 * @param e 
 */
afterQueryOfExport(e: AfterQueryOfExportEvent): void {
    // 按模板导出时，删除每个采购申请单中金额小于500的分录行 

    // 获取待导出的采购申请单
    let queryValues = e.getQueryValues();
    // 遍历所有采购申请单
    for (let queryValue of queryValues) {
        let rows = queryValue.getDynamicObjectCollection("entryentity");
        // 遍历分录行
        for (let i = 0; i < rows.size(); i++) {
            let row = rows.get(i);
            if (row.get("kdtest_amount") != null && row.get("kdtest_amount").compareTo(new BigDecimal(500)) < 0) {
                // 移除金额小于500的分录行
                rows.remove(row);
            }
        }
    }
    e.setQueryValues(queryValues);
}
```
