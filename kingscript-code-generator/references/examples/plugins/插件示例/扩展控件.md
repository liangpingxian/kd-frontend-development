# 扩展控件 Tips 支持案例集（ITipsSupport）

> **注意**：`ITipsSupport` 接口不直接对插件开放，由自定义控件内部实现以支持 Tips 提示功能。以下示例演示的是在表单插件中获取已实现该接口的控件，调用其 Tips 相关方法。

## showFormByTips - 通过 Tips 打开表单

**所属接口**：`ITipsSupport`
**签名**：`showFormByTips(formId: string, elementId: string, triggerType: string): void`
**典型用途**：在控件旁边以 Tips 气泡的形式弹出一个表单页面，常用于字段旁展示关联信息卡片（如供应商卡片、物料详情卡片）。

### 业务场景：鼠标悬停供应商字段时弹出供应商信息卡片

采购订单中，用户将鼠标移到供应商字段上方时，以 Tips 气泡形式弹出供应商的基本信息卡片（联系人、信用额度、合作年限等），无需离开当前页面。

```typescript
import { AbstractBillPlugIn } from "@cosmic/bos-core/kd/bos/bill";
import { BeforeDoOperationEventArgs } from "@cosmic/bos-core/kd/bos/form/events";
import { FormOperate } from "@cosmic/bos-core/kd/bos/form/operate";
import { DynamicObject } from "@cosmic/bos-core/kd/bos/dataentity/entity";

class PurOrderFormPlugin extends AbstractBillPlugIn {

  registerListener(e: $.java.util.EventObject): void {
    let supplierEdit = this.getView().getControl("kdec_supplier") as any;
    supplierEdit.addTipsListener(this);
  }

  beforeShowTips(event: any): void {
    let supplierObj = this.getModel().getValue("kdec_supplier") as DynamicObject;
    if (supplierObj == null) {
      return;
    }

    // 通过 Tips 气泡打开供应商信息卡片表单
    // formId: 卡片表单标识
    // elementId: 触发 Tips 的控件标识（Tips 气泡锚定到该控件旁）
    // triggerType: 触发方式，如 "hover" 或 "click"
    let tipsControl = this.getView().getControl("kdec_supplier") as any;
    tipsControl.showFormByTips("kdec_supplier_card", "kdec_supplier", "hover");
  }
}

let plugin = new PurOrderFormPlugin();
export { plugin };
```

**要点说明**：

1. `formId`：要在 Tips 气泡中打开的表单标识，需预先在设计器中创建该卡片表单
2. `elementId`：气泡锚定的控件标识，Tips 会显示在该控件附近
3. `triggerType`：触发类型，常见值为 `"hover"`（悬停触发）和 `"click"`（点击触发）
4. 卡片表单中可通过页面参数获取当前供应商 ID，加载并展示详细信息

## addTipsListener - 注册 Tips 监听器

**所属接口**：`ITipsSupport`
**签名**：`addTipsListener(listener: TipsListener): void`
**典型用途**：为支持 Tips 的控件注册 `TipsListener`，使插件能在 Tips 显示前后执行自定义逻辑（如动态生成提示内容、记录用户浏览行为）。

### 业务场景：金额字段 Tips 动态显示汇率换算结果

付款单的金额字段注册 Tips 监听，用户悬停时动态计算并展示本位币和外币的换算结果。

```typescript
import { AbstractBillPlugIn } from "@cosmic/bos-core/kd/bos/bill";
import { DynamicObject } from "@cosmic/bos-core/kd/bos/dataentity/entity";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class PayBillFormPlugin extends AbstractBillPlugIn {

  registerListener(e: $.java.util.EventObject): void {
    // 为金额字段注册 Tips 监听
    let amountField = this.getView().getControl("kdec_pay_amount") as any;
    amountField.addTipsListener(this);
  }

  beforeShowTips(event: any): void {
    let amount = this.getModel().getValue("kdec_pay_amount") as BigDecimal;
    let currencyObj = this.getModel().getValue("kdec_currency") as DynamicObject;

    if (amount == null || currencyObj == null) {
      event.setTipsContent("请先填写金额和币种");
      return;
    }

    let currencyName = currencyObj.getString("name");

    // 查询当日汇率
    let rateObj = QueryServiceHelper.queryOne(
      "kdec_exchange_rate",
      "kdec_rate",
      [
        new QFilter("kdec_currency.id", QCP.equals, currencyObj.getLong("id")),
        new QFilter("kdec_effective_date", QCP.less_equals, new Date())
      ],
      "kdec_effective_date desc"
    );

    if (rateObj != null) {
      let rate = rateObj.get("kdec_rate") as BigDecimal;
      let localAmount = amount.multiply(rate).setScale(2, BigDecimal.ROUND_HALF_UP);
      let tip = currencyName + " " + amount.toPlainString()
        + "\n汇率：" + rate.toPlainString()
        + "\n折合本位币：" + localAmount.toPlainString() + " CNY";
      event.setTipsContent(tip);
    } else {
      event.setTipsContent(currencyName + " " + amount.toPlainString() + "\n（未查到汇率）");
    }
  }

  afterShowTips(event: any): void {
    // Tips 显示后的回调，可用于埋点统计
  }
}

let plugin = new PayBillFormPlugin();
export { plugin };
```

**要点说明**：

1. `addTipsListener(this)` 将插件注册为 Tips 监听器，需实现 `beforeShowTips` 和 `afterShowTips`
2. 在 `beforeShowTips` 中通过 `event.setTipsContent(text)` 动态设置提示内容
3. Tips 内容支持换行（`\n`），适合展示多行摘要信息
4. 不是所有控件都实现了 `ITipsSupport`，调用前需确认控件类型

## hideTips - 隐藏 Tips 提示

**所属接口**：`ITipsSupport`
**签名**：`hideTips(): void`
**典型用途**：主动关闭当前显示的 Tips 气泡，常用于业务条件变化后自动收起提示、或在执行操作后清除提示信息。

### 业务场景：选择供应商后自动隐藏"请选择供应商"的 Tips 提示

采购申请单初始打开时，供应商字段显示一个 Tips 引导提示"请选择供应商"。用户选择供应商后，自动隐藏该提示。

```typescript
import { AbstractBillPlugIn } from "@cosmic/bos-core/kd/bos/bill";
import { PropertyChangedArgs } from "@cosmic/bos-core/kd/bos/entity/datamodel/events";

class PurReqFormPlugin extends AbstractBillPlugIn {

  afterBindData(e: $.java.util.EventObject): void {
    super.afterBindData(e);

    // 供应商为空时显示引导提示
    let supplier = this.getModel().getValue("kdec_supplier");
    if (supplier == null) {
      let supplierEdit = this.getView().getControl("kdec_supplier") as any;
      let tips: any = {};
      tips.content = "请选择供应商后再填写明细";
      tips.type = "info";
      supplierEdit.addTips(tips);
    }
  }

  propertyChanged(e: PropertyChangedArgs): void {
    super.propertyChanged(e);

    if (e.getProperty().getName() === "kdec_supplier") {
      let newValue = e.getChangeSet()[0].getNewValue();
      if (newValue != null) {
        // 供应商已选择，主动隐藏 Tips
        let supplierEdit = this.getView().getControl("kdec_supplier") as any;
        supplierEdit.hideTips();
      }
    }
  }
}

let plugin = new PurReqFormPlugin();
export { plugin };
```

**要点说明**：

1. `hideTips()` 立即关闭当前控件上显示的 Tips 气泡
2. 通常与 `addTips` 配合使用：先添加提示，条件满足后隐藏
3. 调用 `hideTips()` 后，若未重新 `addTips`，后续不会再自动弹出

## addTips - 添加 Tips 提示

**所属接口**：`ITipsSupport`
**签名**：`addTips(tips: Tips): void`
**典型用途**：在控件旁主动添加一个 Tips 提示气泡，用于业务引导、数据校验提示、状态说明等。

### 业务场景：分录行金额异常时在对应字段旁显示警告 Tips

销售订单保存前检查分录行，对单价为零或金额超上限的行，在对应字段旁添加黄色警告 Tips 提示。

```typescript
import { AbstractBillPlugIn } from "@cosmic/bos-core/kd/bos/bill";

class SalOrderFormPlugin extends AbstractBillPlugIn {

  beforeDoOperation(e: BeforeDoOperationEventArgs): void {
    super.beforeDoOperation(e);

    let operate = e.getSource() as FormOperate;
    let operateKey = operate.getOperateKey();
    if (operateKey !== "save") {
      return;
    }

    let rowCount = this.getModel().getEntryRowCount("kdec_entryentity");
    let hasWarning = false;

    for (let i = 0; i < rowCount; i++) {
      let price = this.getModel().getValue("kdec_price", i) as BigDecimal;
      let amount = this.getModel().getValue("kdec_amount", i) as BigDecimal;
      let maxLimit = new BigDecimal("1000000");

      // 单价为零警告
      if (price == null || price.compareTo(BigDecimal.ZERO) === 0) {
        let priceField = this.getView().getControl("kdec_price") as any;
        let tips: any = {};
        tips.content = "第" + (i + 1) + "行单价为零，请确认是否为赠品";
        tips.type = "warning";
        priceField.addTips(tips);
        hasWarning = true;
      }

      // 金额超限警告
      if (amount != null && amount.compareTo(maxLimit) > 0) {
        let amountField = this.getView().getControl("kdec_amount") as any;
        let tips: any = {};
        tips.content = "第" + (i + 1) + "行金额超过100万，需额外审批";
        tips.type = "warning";
        amountField.addTips(tips);
        hasWarning = true;
      }
    }

    if (hasWarning) {
      this.getView().showTipNotification("存在异常数据，请查看字段旁的提示信息");
    }
  }
}

let plugin = new SalOrderFormPlugin();
export { plugin };
```

**要点说明**：

1. Tips 对象的常用属性：`content`（提示文本）、`type`（类型：`info`/`warning`/`error`）
2. 同一控件可多次调用 `addTips`，新 Tips 会替换旧的（非叠加）
3. 添加后的 Tips 持续显示直到调用 `hideTips()` 或页面刷新
4. Tips 适合轻量级引导提示，重要校验仍应使用 `showErrorNotification` 或校验器阻断操作
