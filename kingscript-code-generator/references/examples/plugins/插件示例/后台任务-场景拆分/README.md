# 后台任务-场景拆分

这个目录聚焦“怎么从业务页面动态发起调度”的问题，优先沉淀表单按钮点击后创建作业、生成计划的完整链路，而不是只讲后台任务类本身怎么写。

## 案例列表

- [`createPlanAndDispatchJob.md`](./createPlanAndDispatchJob.md): 页面点击后动态创建作业并生成一次性计划

## 使用建议

- 如果你遇到的是“页面按钮触发后怎么创建调度计划”，先看这里。
- 如果你已经处在 `AbstractTask` 的 `execute`、`stop`、消息上报场景，回到 [`后台任务.md`](../后台任务.md) 看任务执行侧能力。
