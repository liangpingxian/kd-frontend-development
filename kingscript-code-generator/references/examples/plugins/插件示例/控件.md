## AfterAddControlListener

### afterAddControl

**签名**：`afterAddControl(event: AfterAddControlEvent): void`
**触发控件**：FormDesigner / AbstractGrid
**场景**：表单设计器动态添加控件后，自动设置新增控件的默认属性。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let grid = this.getView().getControl("kdec_entrygrid") as any;
  grid.addAfterAddControlListener(this);
}

afterAddControl(event: any): void {
  // 新增控件后，自动设置默认宽度和锁定性
  let controlKey = event.getControlKey();
  if (controlKey != null) {
    this.getView().setEnable(false, controlKey);
    this.getView().showTipNotification("已添加控件：" + controlKey);
  }
}
```

## CellClickListener

### cellClick

**签名**：`cellClick(evt: CellClickEvent): void`
**触发控件**：Button、表单控件基类
**场景**：单据体中的按钮列，点击后弹出对应行的物料详情。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let grid = this.getView().getControl("kdec_entrygrid") as any;
  grid.addCellClickListener(this);
}

cellClick(evt: any): void {
  let fieldKey = evt.getFieldKey();
  let row = evt.getRow();

  if (fieldKey === "kdec_btn_detail") {
    let materialObj = this.getModel().getValue("kdec_material", row) as DynamicObject;
    if (materialObj != null) {
      let param = new BillShowParameter();
      param.setFormId("bd_material");
      param.setPkId(materialObj.get("id"));
      param.getOpenStyle().setShowType(ShowType.Modal);
      this.getView().showForm(param);
    }
  }
}
```

### cellDoubleClick

**签名**：`cellDoubleClick(evt: CellClickEvent): void`
**触发控件**：AbstractGrid
**说明**：不推荐使用，建议使用 `cellClick` 替代。
**场景**：双击单据体单元格，快速打开行编辑弹窗。

```typescript
cellDoubleClick(evt: any): void {
  let fieldKey = evt.getFieldKey();
  let row = evt.getRow();

  if (fieldKey === "kdec_remark") {
    // 双击备注列，弹出大文本编辑窗口
    let currentValue = this.getModel().getValue("kdec_remark", row) as string;
    let param = new FormShowParameter();
    param.setFormId("kdec_text_editor");
    param.getOpenStyle().setShowType(ShowType.Modal);
    param.setCustomParam("initText", currentValue || "");
    param.setCustomParam("rowIndex", row);
    param.setCloseCallBack(new CloseCallBack(plugin, "editRemark"));
    this.getView().showForm(param);
  }
}
```

## ClickListener

### beforeClick

**签名**：`beforeClick(evt: BeforeClickEvent): void`
**触发控件**：Button、TextEdit（按钮+文本编辑风格）
**场景**：点击"查看源单"按钮前，校验源单编号是否为空。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let btn = this.getView().getControl("kdec_btn_viewsource") as any;
  btn.addClickListener(this);
}

beforeClick(evt: any): void {
  if (evt.getSource() instanceof Button && evt.getSource().getKey() === "kdec_btn_viewsource") {
    let sourceNo = this.getModel().getValue("kdec_source_billno") as string;
    if (sourceNo == null || sourceNo === "") {
      this.getView().showTipNotification("源单编号为空，无法查看");
      evt.setCancel(true);
    }
  }
}
```

**注意**：必须重写 `beforeClick`，即使方法体为空也需声明，否则可能报异常。

### click

**签名**：`click(evt: EventObject): void`
**触发控件**：Button、Container
**场景**：点击"查看源单"按钮，打开关联的采购订单详情页。

```typescript
click(evt: any): void {
  if (evt.getSource() instanceof Button) {
    let btn = evt.getSource() as Button;
    if (btn.getKey() === "kdec_btn_viewsource") {
      let sourceNo = this.getModel().getValue("kdec_source_billno") as string;
      let obj = BusinessDataServiceHelper.loadSingle(
        "pm_purorderbill",
        [new QFilter("billno", QCP.equals, sourceNo)]
      );
      if (obj != null) {
        let param = new BillShowParameter();
        param.setFormId("pm_purorderbill");
        param.setPkId(obj.get("id"));
        param.getOpenStyle().setShowType(ShowType.Modal);
        this.getView().showForm(param);
      } else {
        this.getView().showErrorNotification("未找到源单：" + sourceNo);
      }
    }
  }
}
```

**场景**：容器面板显示时，根据业务类型动态控制内部字段可见性。

```typescript
  }
}
```

## EnterKeyListener

### enterKeyEvent

**签名**：`enterKeyEvent(event: EnterKeyEvent): void`
**触发控件**：输入框
**场景**：物料编码输入框回车后，自动查询并填充物料名称和规格。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let textEdit = this.getView().getControl("kdec_material_code") as any;
  textEdit.addEnterKeyListener(this);
}

enterKeyEvent(event: any): void {
  let inputValue = this.getModel().getValue("kdec_material_code") as string;
  if (inputValue != null && inputValue !== "") {
    let materialObj = QueryServiceHelper.queryOne(
      "bd_material",
      "id,name,modelnum",
      [new QFilter("number", QCP.equals, inputValue)]
    );
    if (materialObj != null) {
      this.getModel().setValue("kdec_material_name", materialObj.getString("name"));
      this.getModel().setValue("kdec_model", materialObj.getString("modelnum"));
    } else {
      this.getView().showTipNotification("未找到编码为 " + inputValue + " 的物料");
    }
  }
}
```

## EntryFilterChangedListener

### entryFilterChanged

**签名**：`entryFilterChanged(e: EntryFilterChangedEvent): void`
**触发控件**：EntryGrid（表格过滤条件改变）
**场景**：用户修改单据体表格的过滤条件后，重新统计过滤后的合计金额并更新表头。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let entryGrid = this.getView().getControl("kdec_entrygrid") as any;
  entryGrid.addEntryFilterChangedListener(this);
}

entryFilterChanged(e: $.kd.bos.form.control.events.EntryFilterChangedEvent): void {
  // 过滤条件变更后，重算过滤可见行的合计金额
  let totalAmount = BigDecimal.ZERO;
  let rowCount = this.getModel().getEntryRowCount("kdec_entryentity");

  for (let i = 0; i < rowCount; i++) {
    let amt = this.getModel().getValue("kdec_amount", i) as BigDecimal;
    if (amt != null) {
      totalAmount = totalAmount.add(amt);
    }
  }

  this.getModel().setValue("kdec_filter_total", totalAmount);
  this.getView().showTipNotification("过滤后合计：" + totalAmount.toPlainString());
}
```

## EntryGridColAllCheckedListener

### entryGridColAllChecked

**签名**：`entryGridColAllChecked(e: EntryGridColAllCheckedEvent): void`
**触发控件**：EntryGrid（表格列头勾选全部）
**场景**：用户勾选"是否赠品"列头全选时，自动将所有行的单价置零。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let entryGrid = this.getView().getControl("kdec_entrygrid") as any;
  entryGrid.addEntryGridColAllCheckedListener(this);
}

entryGridColAllChecked(e: $.kd.bos.form.control.events.EntryGridColAllCheckedEvent): void {
  let fieldKey = e.getFieldKey();
  let checked = e.isChecked();

  if (fieldKey === "kdec_is_gift" && checked) {
    let rowCount = this.getModel().getEntryRowCount("kdec_entryentity");
    for (let i = 0; i < rowCount; i++) {
      this.getModel().setValue("kdec_price", BigDecimal.ZERO, i);
      this.getModel().setValue("kdec_is_gift", true, i);
    }
    this.getView().showTipNotification("已将所有行标记为赠品，单价已清零");
  }
}
```

## EntryGridSetRowDataListener

### entryGridSetRowData

**签名**：`entryGridSetRowData(e: EntryGridSetRowDataEvent): void`
**触发控件**：EntryGrid（根据行数据赋值）
**场景**：表格渲染每行数据时，根据库存余额动态标红库存不足的行。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let entryGrid = this.getView().getControl("kdec_entrygrid") as any;
  entryGrid.addEntryGridSetRowDataListener(this);
}

entryGridSetRowData(e: $.kd.bos.form.control.events.EntryGridSetRowDataEvent): void {
  let rowData = e.getRowData();
  let rowIndex = e.getRowIndex();

  let qty = this.getModel().getValue("kdec_qty", rowIndex) as BigDecimal;
  let stockQty = this.getModel().getValue("kdec_stock_qty", rowIndex) as BigDecimal;

  // 需求数量超过库存时，设置行背景色为红色警告
  if (qty != null && stockQty != null && qty.compareTo(stockQty) > 0) {
    rowData.put("kdec_qty_color", "red");
  }
}
```

## ItemClickListener

### beforeItemClick

**签名**：`beforeItemClick(evt: BeforeItemClickEvent): void`
**触发控件**：Toolbar
**场景**：点击"删除"菜单前，校验单据是否已被引用，已引用则阻止删除。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let toolbar = this.getView().getControl("tbmain") as Toolbar;
  toolbar.addItemClickListener(this);
}

beforeItemClick(evt: any): void {
  if (evt.getItemKey() === "kdec_delete") {
    let billNo = this.getModel().getValue("billno") as string;
    let referenced = QueryServiceHelper.exists(
      "im_purinstorebill",
      [new QFilter("kdec_source_billno", QCP.equals, billNo)]
    );
    if (referenced) {
      this.getView().showErrorNotification("该单据已被入库单引用，不允许删除");
      evt.setCancel(true);
    }
  }
}
```

**注意**：`ItemKey` 字符串必须与菜单标识完全一致（区分大小写），写错会导致不进入预期逻辑。

### itemClick

```typescript
  }
}
```

## OCRListener

### processOCR

**签名**：`processOCR(evt: OCREvent): void`
**触发控件**：OCR 控件
**场景**：发票 OCR 识别后，将识别结果中的发票号、金额、日期写入表单字段。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let ocrControl = this.getView().getControl("kdec_ocr") as any;
  ocrControl.addOCRListener(this);
}

processOCR(evt: any): void {
  let result = evt.getOCRResult();
  if (result != null) {
    let invoiceNo = result.getString("invoiceNo");
    let amount = result.getString("amount");
    let invoiceDate = result.getString("invoiceDate");

    if (invoiceNo != null) {
      this.getModel().setValue("kdec_invoice_no", invoiceNo);
    }
    if (amount != null) {
      this.getModel().setValue("kdec_invoice_amount", new BigDecimal(amount));
    }
    if (invoiceDate != null) {
      this.getModel().setValue("kdec_invoice_date", invoiceDate);
    }
  }
}
```

### finishOCR

**签名**：`finishOCR(): void`
**场景**：OCR 识别全部完成后，自动触发发票验真校验。

```typescript
finishOCR(): void {
  let invoiceNo = this.getModel().getValue("kdec_invoice_no") as string;
  if (invoiceNo != null && invoiceNo !== "") {
    // 查询发票是否已录入过（防重复）
    let exists = QueryServiceHelper.exists(
      "kdec_invoice_bill",
      [new QFilter("kdec_invoice_no", QCP.equals, invoiceNo)]
    );
    if (exists) {
      this.getView().showErrorNotification("发票 " + invoiceNo + " 已存在，请勿重复录入");
    } else {
      this.getView().showSuccessNotification("OCR 识别完成，发票信息已填入");
    }
  }
}
```

### notifying

**签名**：`notifying(status: number, msg: string): void`
**场景**：OCR 处理过程中实时展示状态消息（上传中、识别中、完成）。

```typescript
notifying(status: number, msg: string): void {
  if (status === 0) {
    this.getView().showTipNotification("图片上传中...");
  } else if (status === 1) {
    this.getView().showTipNotification("正在识别：" + msg);
  } else if (status === 2) {
    this.getView().showSuccessNotification("识别完成");
  } else if (status === -1) {
    this.getView().showErrorNotification("识别失败：" + msg);
  }
}
```

## ProgressListener

### onProgress

**签名**：`onProgress(evt: ProgressEvent): void`
**触发控件**：ProgressBar
**场景**：批量导入物料数据时，进度条实时显示处理进度，完成后刷新列表。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let progressBar = this.getView().getControl("kdec_progressbar") as any;
  progressBar.addProgressListener(this);
}

onProgress(evt: any): void {
  let progress = evt.getProgress();
  let total = evt.getTotal();

  if (progress >= total) {
    this.getView().showSuccessNotification("导入完成，共处理 " + total + " 条记录");
    this.getView().invokeOperation("refresh");
  }
}
```

**注意**：
- 使用 `this.getView().getPageCache().put(key, value)` 缓存进度数据（`set` 方法不可用，需用 `put`）
- 进度条执行中如遇中断，排查是否因超时导致

## RateListener

### update (Rate)

**签名**：`update(e: RateEvent): void`
**触发控件**：Rate 评分控件
**场景**：供应商评分变更后，自动更新供应商等级。

```typescript
  registerListener(e: $.java.util.EventObject): void {

update(e: any): void {
  let score = e.getValue() as number;
  let level = "";

  if (score >= 5) {
    level = "A";

}
```

## BaseDataFuzzySearchListener

```typescript
  materialEdit.addBaseDataFuzzySearchListener(this);

handleData(evt: any): void {
  let resultList = evt.getResultList();

  // 过滤掉禁用物料，补充库存信息
  let filteredList = new ArrayList();
  for (let item of resultList) {
    let materialId = item.getLong("id");
    let stockObj = QueryServiceHelper.queryOne(

    );

    item.put("displayText", item.getString("name") + " [库存:" + stockQty.toPlainString() + "]");

  }

  evt.setResultList(filteredList);
}
```

## Before7SelectListener (BaseDataEdit)

### before7Select

**签名**：`before7Select(evt: Before7SelectEvent): void`
**触发控件**：BasedataEdit
**场景**：选择物料前，根据当前单据的仓库过滤只显示该仓库有库存的物料。

```typescript
  registerListener(e: $.java.util.EventObject): void {
  let materialEdit = this.getView().getControl("kdec_material") as BasedataEdit;
  materialEdit.addBeforeF7SelectListener(this);
}

before7Select(evt: any): void {
  if (evt.getProperty().getName() === "kdec_material") {
    let warehouseObj = this.getModel().getValue("kdec_warehouse") as DynamicObject;
    if (warehouseObj != null) {
      let filters = new ArrayList();
      filters.add(new QFilter("enable", QCP.equals, "1"));

      // 只显示当前仓库有库存的物料
      let materialIds = QueryServiceHelper.query(
        "im_inventory_balance",
        "material.id",
        [
          new QFilter("warehouse.id", QCP.equals, warehouseObj.getLong("id")),
          new QFilter("qty", QCP.large_than, BigDecimal.ZERO)
        ]
      );

      let idList = new ArrayList();
      for (let row of materialIds) {
        idList.add(row.get("material.id"));
      }

      if (idList.size() > 0) {
        filters.add(new QFilter("id", QCP.in, idList));
      }

      evt.setCustomQFilters(filters);
    }
  }
}
```
