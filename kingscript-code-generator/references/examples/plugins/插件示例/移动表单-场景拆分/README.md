这个目录聚焦移动表单里“结构化控件怎么配”的问题，优先沉淀 MobTable 这种样例密度最高、复用价值也最高的移动端场景。

## 案例列表

- [`configureMobTableColumnsAndPackageData.md`](./configureMobTableColumnsAndPackageData.md): 初始化 MobTable 列定义，并在返回前打包展示数据

## 使用建议

- 如果你面对的是移动表格列、展示格式、数据打包，先看这里。
- 如果你要处理定位、上传、搜索等其他移动交互，可回到 [`移动表单.md`](../移动表单.md) 和 [`移动控件.md`](../移动控件.md)。
