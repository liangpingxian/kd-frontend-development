## MobileSearchCancelListener

### cancel - 搜索取消输入

**签名**：`cancel(): void`
**触发控件**：MobileSearch
**场景**：移动端物料搜索取消时，恢复分录为全量数据并清除搜索高亮。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";

class MaterialSearchMobPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let mobileSearch = this.getView().getControl("kdec_mob_search") as any;
    mobileSearch.addMobileSearchCancelListener(this);
  }

  cancel(): void {
    // 取消搜索：清除过滤条件，刷新列表显示全量数据
    this.getModel().setValue("kdec_search_keyword", "");
    this.getView().invokeOperation("refresh");
    this.getView().showTipNotification("已退出搜索");
  }
}

let plugin = new MaterialSearchMobPlugin();
export { plugin };
```

**要点说明**：

1. 用户点击搜索栏的"取消"按钮时触发
2. 应在此清除之前的搜索过滤条件，恢复到初始数据状态
3. 无参数，无法获取之前的搜索关键词，需从表单字段中自行管理

## MobileSearchFocusListener

### focus - 搜索聚焦

**签名**：`focus(): void`
**触发控件**：MobileSearch
**场景**：搜索框获得焦点时，加载搜索历史记录供用户快速选择。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { RequestContext } from "@cosmic/bos-core/kd/bos/context";

class OrderSearchMobPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let mobileSearch = this.getView().getControl("kdec_mob_search") as any;
    mobileSearch.addMobileSearchFocusListener(this);
  }

  focus(): void {
    // 搜索框聚焦时，查询当前用户最近10条搜索记录
    let userId = RequestContext.get().getCurrUserId();
    let histories = QueryServiceHelper.query(
      "kdec_search_history",
      "kdec_keyword,kdec_search_time",
      [new QFilter("kdec_user_id", QCP.equals, userId)],
      "kdec_search_time desc", 10
    );

    if (histories.size() > 0) {
      let keywords = new ArrayList();
      for (let row of histories) {
        keywords.add(row.getString("kdec_keyword"));
      }
      // 设置搜索建议列表
      let searchControl = this.getView().getControl("kdec_mob_search") as any;
      searchControl.setSuggestions(keywords);
    }
  }
}

let plugin = new OrderSearchMobPlugin();
export { plugin };
```

## MobileSearchTextChangeListener

### click - 搜索文本输入

**签名**：`click(evt: MobileSearchTextChangeEvent): void`
**触发控件**：MobileSearch
**场景**：用户在搜索框输入文本时实时过滤，按关键词模糊匹配订单编号和客户名称。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class OrderListMobPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let mobileSearch = this.getView().getControl("kdec_mob_search") as any;
    mobileSearch.addMobileSearchTextChangeListener(this);
  }

  click(evt: any): void {
    let keyword = evt.getText();

    if (keyword == null || keyword.length < 2) {
      // 少于2个字符不触发搜索，避免频繁查询
      return;
    }

    // 模糊匹配订单编号或客户名称
    let results = QueryServiceHelper.query(
      "sm_salorder",
      "id,billno,customer.name,totalamount",
      [
        new QFilter("billno", QCP.like, "%" + keyword + "%")
          .or(new QFilter("customer.name", QCP.like, "%" + keyword + "%"))
      ],
      "billno desc", 20
    );

    // 将搜索结果设置到列表控件
    let searchControl = this.getView().getControl("kdec_mob_search") as any;
    let resultList = new ArrayList();
    for (let row of results) {
      let item: any = {};
      item.id = row.get("id");
      item.title = row.getString("billno");
      item.subtitle = row.getString("customer.name");
      resultList.add(item);
    }
    searchControl.setSearchResults(resultList);
  }
}

let plugin = new OrderListMobPlugin();
export { plugin };
```

**要点说明**：

1. 方法名虽为 `click`，实际是文本变更事件，每次输入内容变化都会触发
2. 建议设置最小输入长度（如 2 字符）再触发查询，避免频繁调用
3. 注意 SDK 限流：限制查询结果条数（建议 ≤ 20 条）

## PullRefreshListener

### pullRefresh - 卡片分录下拉刷新

**签名**：`pullRefresh(evt: PullRefreshEvent): void`
**触发控件**：CardEntry（卡片分录）
**场景**：移动端配送单卡片分录下拉刷新，重新加载待配送的订单数据。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { RequestContext } from "@cosmic/bos-core/kd/bos/context";

class DeliveryMobFormPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let cardEntry = this.getView().getControl("kdec_card_entry") as any;
    cardEntry.addPullRefreshListener(this);
  }

  pullRefresh(evt: any): void {
    let userId = RequestContext.get().getCurrUserId();

    // 查询当前配送员的待配送订单
    let orders = QueryServiceHelper.query(
      "kdec_delivery_order",
      "id,kdec_order_no,kdec_customer_name,kdec_address,kdec_delivery_time",
      [
        new QFilter("kdec_deliverer.id", QCP.equals, userId),
        new QFilter("kdec_status", QCP.equals, "pending")
      ],
      "kdec_delivery_time asc", 50
    );

    // 清空并重建分录
    this.getModel().deleteEntryData("kdec_entryentity");

    for (let i = 0; i < orders.size(); i++) {
      let order = orders.get(i);
      let rowIndex = this.getModel().createNewEntryRow("kdec_entryentity");
      this.getModel().setValue("kdec_order_no", order.getString("kdec_order_no"), rowIndex);
      this.getModel().setValue("kdec_customer_name", order.getString("kdec_customer_name"), rowIndex);
      this.getModel().setValue("kdec_address", order.getString("kdec_address"), rowIndex);
    }

    this.getView().showSuccessNotification("已刷新，共 " + orders.size() + " 个待配送订单");
  }
}

let plugin = new DeliveryMobFormPlugin();
export { plugin };
```

**要点说明**：

1. `PullRefreshListener` 作用于 `CardEntry`（卡片分录），与 `IMobFormPlugin.refreshData` 不同
2. 下拉刷新适合重新加载数据量不大的列表，大数据量应使用分页加载

## SearchClickListener

### click - 筛选排序搜索点击

**签名**：`click(evt: SearchClickEvent): void`
**触发控件**：MobFilterSort
**场景**：移动端筛选排序组件中的搜索按钮点击，执行自定义搜索逻辑。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class FilterSortMobPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let filterSort = this.getView().getControl("kdec_mob_filtersort") as any;
    filterSort.addSearchClickListener(this);
  }

  click(evt: any): void {
    let keyword = evt.getSearchText();

    if (keyword == null || keyword === "") {
      this.getView().showTipNotification("请输入搜索关键词");
      return;
    }

    // 在筛选排序面板中搜索供应商
    let suppliers = QueryServiceHelper.query(
      "bd_supplier",
      "id,number,name",
      [
        new QFilter("name", QCP.like, "%" + keyword + "%")
          .or(new QFilter("number", QCP.like, "%" + keyword + "%"))
      ],
      "number", 20
    );

    this.getView().showTipNotification("搜索到 " + suppliers.size() + " 个供应商");
  }
}

let plugin = new FilterSortMobPlugin();
export { plugin };
```

## AfterMobileListPushDownRefreshListener

### afterPushDownRefresh - 列表下拉刷新后

**签名**：`afterPushDownRefresh(evt: AfterMobileListPushDownRefreshEvent): void`
**触发控件**：BillList（移动端列表）
**场景**：移动端单据列表下拉刷新完成后，更新列表顶部的统计信息（待办数量、金额汇总）。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { RequestContext } from "@cosmic/bos-core/kd/bos/context";

class OrderListMobPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let billList = this.getView().getControl("billlistap") as any;
    billList.addAfterMobileListPushDownRefreshListener(this);
  }

  afterPushDownRefresh(evt: any): void {
    let userId = RequestContext.get().getCurrUserId();

    // 刷新后统计待办数据
    let pendingCount = QueryServiceHelper.queryOne(
      "sm_salorder",
      "count(id) as cnt",
      [
        new QFilter("creator.id", QCP.equals, userId),
        new QFilter("billstatus", QCP.equals, "A")
      ]
    );

    let totalAmount = QueryServiceHelper.queryOne(
      "sm_salorder",
      "sum(totalamount) as total",
      [
        new QFilter("creator.id", QCP.equals, userId),
        new QFilter("billstatus", QCP.equals, "A")
      ]
    );

    let cnt = pendingCount != null ? pendingCount.getInt("cnt") : 0;
    let amt = totalAmount != null ? totalAmount.get("total") as BigDecimal : BigDecimal.ZERO;

    this.getView().showTipNotification(
      "待处理 " + cnt + " 单，合计 " + (amt != null ? amt.toPlainString() : "0") + " 元"
    );
  }
}

let plugin = new OrderListMobPlugin();
export { plugin };
```

## IBeforeCreateMobTableColumnsListener

### beforeCreateMobTableColumns - 移动表格创建列前

**签名**：`beforeCreateMobTableColumns(event: BeforeCreateMobTableColumnsEvent): void`
**触发控件**：MobTable
**场景**：移动端表格创建列前，根据用户角色动态隐藏敏感列（成本、利润）。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { RequestContext } from "@cosmic/bos-core/kd/bos/context";

class MobTableColumnPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let mobTable = this.getView().getControl("kdec_mob_table") as any;
    mobTable.addBeforeCreateMobTableColumnsListener(this);
  }

  beforeCreateMobTableColumns(event: any): void {
    let columns = event.getColumns();
    let userId = RequestContext.get().getCurrUserId();

    // 查询用户是否有成本查看权限
    let hasPermission = QueryServiceHelper.exists(
      "kdec_user_permission",
      [
        new QFilter("kdec_user.id", QCP.equals, userId),
        new QFilter("kdec_perm_code", QCP.equals, "view_cost")
      ]
    );

    if (!hasPermission) {
      for (let col of columns) {
        let key = col.getKey();
        if (key === "kdec_cost_price" || key === "kdec_profit") {
          col.setVisible(false);
        }
      }
    }
  }
}

let plugin = new MobTableColumnPlugin();
export { plugin };
```

## IMobTableDataProviderListener

### createMobTableDataProvider - 移动表格数据提供

**签名**：`createMobTableDataProvider(event: MobTableDataProviderEvent): void`
**触发控件**：MobTable
**场景**：为移动端表格提供自定义数据源，从多个单据汇总查询组装表格数据。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { DynamicObject } from "@cosmic/bos-core/kd/bos/dataentity/entity";

class SalesSummaryMobPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let mobTable = this.getView().getControl("kdec_mob_table") as any;
    mobTable.addMobTableDataProviderListener(this);
  }

  createMobTableDataProvider(event: any): void {
    // 自定义数据源：按客户汇总本月销售额
    let monthStart = new Date();
    monthStart.setDate(1);

    let summaryData = QueryServiceHelper.query(
      "sm_salorder",
      "customer.name as custName,sum(totalamount) as totalAmt,count(id) as orderCount",
      [
        new QFilter("bizdate", QCP.large_equals, monthStart),
        new QFilter("billstatus", QCP.equals, "C")
      ],
      "",
      "customer.name"  // group by
    );

    let dataList = new ArrayList();
    for (let row of summaryData) {
      let dataRow = new HashMap();
      dataRow.put("kdec_customer_name", row.getString("custName"));
      dataRow.put("kdec_order_count", row.getInt("orderCount"));
      dataRow.put("kdec_total_amount", row.get("totalAmt"));
      dataList.add(dataRow);
    }

    event.setDataProvider(dataList);
  }
}

let plugin = new SalesSummaryMobPlugin();
export { plugin };
```

## IMobTablePackageDataHandlerListener

### createMobTablePackageDataHandler - 移动表格数据打包

**签名**：`createMobTablePackageDataHandler(event: MobTablePackageDataHandlerEvent): void`
**触发控件**：MobTable
**场景**：移动表格数据打包发送到前端前，对金额列格式化显示并添加状态标签颜色。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";

class MobTablePackagePlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let mobTable = this.getView().getControl("kdec_mob_table") as any;
    mobTable.addMobTablePackageDataHandlerListener(this);
  }

  createMobTablePackageDataHandler(event: any): void {
    let rows = event.getDataRows();

    for (let row of rows) {
      // 金额格式化：千分位 + 2位小数
      let amount = row.get("kdec_amount") as BigDecimal;
      if (amount != null) {
        let formatted = amount.setScale(2, BigDecimal.ROUND_HALF_UP).toPlainString();
        row.put("kdec_amount_display", formatted + " 元");
      }

      // 状态字段添加颜色标签
      let status = row.get("kdec_status") as string;
      if (status === "pending") {
        row.put("kdec_status_display", "待处理");
        row.put("kdec_status_color", "#FF9800");
      } else if (status === "completed") {
        row.put("kdec_status_display", "已完成");
        row.put("kdec_status_color", "#4CAF50");
      } else if (status === "overdue") {
        row.put("kdec_status_display", "已逾期");
        row.put("kdec_status_color", "#F44336");
      }

      // 超期天数计算
      let dueDate = row.get("kdec_due_date") as Date;
      if (dueDate != null) {
        let today = new Date();
        let diffDays = Math.floor((today.getTime() - dueDate.getTime()) / (1000 * 60 * 60 * 24));
        if (diffDays > 0) {
          row.put("kdec_overdue_days", "逾期 " + diffDays + " 天");
        }
      }
    }
  }
}

let plugin = new MobTablePackagePlugin();
export { plugin };
```

## MobFilterSortInitListener

### mobFilterSortInit - 筛选排序初始化

**签名**：`mobFilterSortInit(evt: MobFilterSortInitEvent): void`
**触发控件**：MobFilterSort
**场景**：移动端筛选排序面板初始化时，预设默认筛选条件和排序规则。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { RequestContext } from "@cosmic/bos-core/kd/bos/context";

class MobFilterInitPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let filterSort = this.getView().getControl("kdec_mob_filtersort") as any;
    filterSort.addMobFilterSortInitListener(this);
  }

  mobFilterSortInit(evt: any): void {
    // 设置默认筛选项
    let filterItems = evt.getFilterItems();

    // 状态筛选：默认选中"待处理"
    for (let item of filterItems) {
      if (item.getKey() === "kdec_status") {
        item.setDefaultValue("pending");
      }
    }

    // 日期筛选：默认近7天
    for (let item of filterItems) {
      if (item.getKey() === "kdec_date_range") {
        let today = new Date();
        let sevenDaysAgo = new Date();
        sevenDaysAgo.setDate(today.getDate() - 7);
        item.setDefaultValue(sevenDaysAgo, today);
      }
    }

    // 排序：默认按日期降序
    let sortItems = evt.getSortItems();
    for (let item of sortItems) {
      if (item.getKey() === "kdec_bizdate") {
        item.setDefaultOrder("desc");
      }
    }
  }
}

let plugin = new MobFilterInitPlugin();
export { plugin };
```

## MobileSearchInitListener

### mobileSearchInit - 搜索控件初始化

**签名**：`mobileSearchInit(evt: MobileSearchInitEvent): void`
**触发控件**：MobileSearch
**场景**：移动端搜索控件初始化时，设置搜索占位符文本、搜索范围和热门搜索词。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class MobSearchInitPlugin extends AbstractMobFormPlugin {

  registerListener(e: $.java.util.EventObject): void {
    let mobileSearch = this.getView().getControl("kdec_mob_search") as any;
    mobileSearch.addMobileSearchInitListener(this);
  }

  mobileSearchInit(evt: any): void {
    // 设置搜索框占位符
    evt.setPlaceholder("输入物料编码或名称搜索");

    // 设置热门搜索词
    let hotKeywords = QueryServiceHelper.query(
      "kdec_search_history",
      "kdec_keyword,count(id) as cnt",
      [new QFilter("kdec_search_time", QCP.large_equals, this.getWeekStart())],
      "",
      "kdec_keyword"  // group by
    );

    let hotWords = new ArrayList();
    let count = 0;
    for (let row of hotKeywords) {
      if (count >= 6) {
        break;
      }
      hotWords.add(row.getString("kdec_keyword"));
      count++;
    }

    if (hotWords.size() > 0) {
      evt.setHotKeywords(hotWords);
    }

    // 设置搜索防抖延迟（毫秒）
    evt.setDebounceDelay(300);
  }

  getWeekStart(): Date {
    let d = new Date();
    d.setDate(d.getDate() - 7);
    return d;
  }
}

let plugin = new MobSearchInitPlugin();
export { plugin };
```

**要点说明**：

1. `mobileSearchInit` 在搜索控件创建时触发一次，用于一次性初始化配置
2. 设置防抖延迟可避免用户快速输入时频繁触发搜索
3. 热门搜索词从搜索历史中按频次统计，提升用户搜索效率
