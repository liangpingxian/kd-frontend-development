## 一、AbstractReportListDataPlugin（报表查询插件）

> **基类**：`AbstractReportListDataPlugin`
> **导入路径**：`@cosmic/bos-core/kd/bos/entity/report`
> **作用**：控制报表的取数逻辑，包括查询、导出、动态列等

### 1. query - 查询数据

**方法签名**：`query(queryParam: any, selectedObj: any): any`

**触发时机**：报表刷新/查询时，由 `ReportServiceImpl.doPluginQuery` 调用

**业务场景**：销售订单利润分析报表 —— 按客户维度统计订单金额、成本、利润率

```typescript
import { AbstractReportListDataPlugin } from "@cosmic/bos-core/kd/bos/entity/report";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class SalesProfitReportDataPlugin extends AbstractReportListDataPlugin {

  /**
   * 查询报表数据
   * @param queryParam 报表查询参数，包含过滤面板条件
   * @param selectedObj 左树选中的节点（无左树时为 null）
   */
  query(queryParam: any, selectedObj: any): any {
    // 从过滤面板获取查询条件
    let filters: QFilter[] = [];
    let filterItems = queryParam.getFilter().getFilterItems();
    for (let item of filterItems) {
      let propName = item.getPropName();
      if ("kdec_startdate" === propName) {
        filters.push(new QFilter("bizdate", QCP.ge, item.getValue()));
      } else if ("kdec_enddate" === propName) {
        filters.push(new QFilter("bizdate", QCP.le, item.getValue()));
      } else if ("kdec_customer.id" === propName) {
        filters.push(new QFilter("customer", QCP.in, item.getValue()));
      }
    }

    // 使用 queryDataSet 查询销售订单明细
    let dataSet = QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "sm_salorder",
      "customer as kdec_customer, "
        + "customer.name as kdec_customer_name, "
        + "salorderentry.amount as kdec_amount, "
        + "salorderentry.costamount as kdec_cost",
      filters.length > 0 ? filters : null,
      null
    );

    // 按客户分组统计
    let resultDataSet = dataSet.copy()
      .groupBy(["kdec_customer", "kdec_customer_name"])
      .sum("kdec_amount", "kdec_total_amount")
      .sum("kdec_cost", "kdec_total_cost")
      .count("kdec_order_count")
      .finish();

    return resultDataSet;
  }
}

let plugin = new SalesProfitReportDataPlugin();
export { plugin };
```

**要点**：
- `queryParam.getFilter().getFilterItems()` 获取过滤面板的过滤条件
- 使用 `QueryServiceHelper.queryDataSet` 返回 DataSet，便于后续分组/join/union
- DataSet 支持链式操作：`groupBy` → `sum`/`count` → `finish`

### 2. export - 导出查询结果数据

**方法签名**：`export(queryParam: any, selectedObj: any): any`

**触发时机**：报表导出 Excel 时，由 `NormalExcelExport.doQueryExportData` 调用

**业务场景**：库存余额报表导出 —— 分页显示但导出时全量导出，且导出数据需补充批次号信息

```typescript
import { AbstractReportListDataPlugin } from "@cosmic/bos-core/kd/bos/entity/report";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class InventoryBalanceExportPlugin extends AbstractReportListDataPlugin {

  /**
   * 报表页面查询（分页显示）
   */
  query(queryParam: any, selectedObj: any): any {
    let filters = this.buildFilters(queryParam);

    let dataSet = QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "im_inventory",
      "material.name as kdec_material_name, "
        + "warehouse.name as kdec_warehouse_name, "
        + "baseqty as kdec_qty, "
        + "baseamount as kdec_amount",
      filters.length > 0 ? filters : null,
      "material.name"
    );

    return dataSet;
  }

  /**
   * 导出时全量查询，补充批次号
   * 分页显示但全量导出的场景
   */
  export(queryParam: any, selectedObj: any): any {
    let filters = this.buildFilters(queryParam);

    // 导出时查询更多字段，包含批次号
    let dataSet = QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "im_inventory",
      "material.name as kdec_material_name, "
        + "material.number as kdec_material_number, "
        + "warehouse.name as kdec_warehouse_name, "
        + "lotnumber as kdec_lot_number, "
        + "baseqty as kdec_qty, "
        + "baseamount as kdec_amount",
      filters.length > 0 ? filters : null,
      "material.name"
    );

    return dataSet;
  }

  /**
   * 构造过滤条件（复用逻辑）
   */
  private buildFilters(queryParam: any): QFilter[] {
    let filters: QFilter[] = [];
    let filterItems = queryParam.getFilter().getFilterItems();
    for (let item of filterItems) {
      if ("kdec_warehouse.id" === item.getPropName()) {
        filters.push(new QFilter("warehouse", QCP.in, item.getValue()));
      } else if ("kdec_material.id" === item.getPropName()) {
        filters.push(new QFilter("material", QCP.in, item.getValue()));
      }
    }
    return filters;
  }
}

let plugin = new InventoryBalanceExportPlugin();
export { plugin };
```

**要点**：
- `export` 在导出时调用，`query` 在页面显示时调用
- 导出可返回比页面显示更多的字段或不同的数据结构
- 公共过滤逻辑抽取为 private 方法避免重复

### 3. exportWithSheet - 分 Sheet 页导出数据

**方法签名**：`exportWithSheet(queryParam: any, selectedObj: any): any`

**触发时机**：导出 Excel 时，支持将数据按条件拆分到多个 Sheet 页

**业务场景**：按仓库维度导出库存报表 —— 每个仓库一个 Sheet 页

```typescript
import { AbstractReportListDataPlugin } from "@cosmic/bos-core/kd/bos/entity/report";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { ArrayList } from "@cosmic/bos-script/java/util";

class WarehouseInventoryExportPlugin extends AbstractReportListDataPlugin {

  query(queryParam: any, selectedObj: any): any {
    return this.queryInventoryData(queryParam, null);
  }

  /**
   * 分 Sheet 页导出数据
   * 每个仓库的数据导出到一个独立的 Sheet 页
   * @returns ReportExportDataResult 列表
   */
  exportWithSheet(queryParam: any, selectedObj: any): any {
    let resultList = new ArrayList();

    // 先查询所有涉及的仓库
    let warehouseDataSet = QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "im_inventory",
      "warehouse as kdec_wh_id, warehouse.name as kdec_wh_name",
      null, null
    );
    let warehouseList = warehouseDataSet.copy()
      .groupBy(["kdec_wh_id", "kdec_wh_name"])
      .finish();

    // 遍历每个仓库，生成独立 Sheet 数据
    while (warehouseList.hasNext()) {
      warehouseList.next();
      let whId = warehouseList.getLong("kdec_wh_id");
      let whName = warehouseList.getString("kdec_wh_name");

      let sheetData = this.queryInventoryData(queryParam, whId);

      // 构造 Sheet 导出结果
      let exportResult = new Object();
      exportResult["sheetName"] = whName;
      exportResult["dataSet"] = sheetData;
      resultList.add(exportResult);
    }

    return resultList;
  }

  /**
   * 查询库存数据
   */
  private queryInventoryData(queryParam: any, warehouseId: any): any {
    let filters: QFilter[] = [];

    if (warehouseId != null) {
      filters.push(new QFilter("warehouse", QCP.equals, warehouseId));
    }

    let filterItems = queryParam.getFilter().getFilterItems();
    for (let item of filterItems) {
      if ("kdec_material.id" === item.getPropName()) {
        filters.push(new QFilter("material", QCP.in, item.getValue()));
      }
    }

    return QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "im_inventory",
      "material.name as kdec_material_name, "
        + "warehouse.name as kdec_warehouse_name, "
        + "baseqty as kdec_qty, "
        + "baseamount as kdec_amount",
      filters.length > 0 ? filters : null,
      "material.name"
    );
  }
}

let plugin = new WarehouseInventoryExportPlugin();
export { plugin };
```

**要点**：
- `exportWithSheet` 返回一个 List，每个元素包含 Sheet 名和对应的 DataSet
- 当 `exportWithSheet` 存在时，优先调用该方法；不存在时回退到 `export`
- 适合按组织、仓库、部门等维度分 Sheet 导出

### 4. getColumns - 改变列信息

**方法签名**：`getColumns(columns: any): any`

**触发时机**：报表查询时，在 `ReportServiceImpl.doPluginQuery` 中调用，用于动态调整报表列

**业务场景**：按月份动态生成报表列 —— 根据查询的日期范围自动生成月度列

```typescript
import { AbstractReportListDataPlugin } from "@cosmic/bos-core/kd/bos/entity/report";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { ArrayList } from "@cosmic/bos-script/java/util";

class MonthlyAmountReportDataPlugin extends AbstractReportListDataPlugin {

  /**
   * 动态修改报表列
   * 根据查询日期范围，生成对应月份的金额列
   */
  getColumns(columns: any): any {
    let queryParam = this.getQueryParam();
    if (queryParam == null) {
      return columns;
    }

    // 从查询参数获取日期范围
    let filterItems = queryParam.getFilter().getFilterItems();
    let startDate: Date = null;
    let endDate: Date = null;
    for (let item of filterItems) {
      if ("kdec_startdate" === item.getPropName()) {
        startDate = item.getValue() as Date;
      } else if ("kdec_enddate" === item.getPropName()) {
        endDate = item.getValue() as Date;
      }
    }

    if (startDate == null || endDate == null) {
      return columns;
    }

    // 生成每个月的列
    let current = new Date(startDate.getFullYear(), startDate.getMonth(), 1);
    let end = new Date(endDate.getFullYear(), endDate.getMonth(), 1);

    while (current <= end) {
      let year = current.getFullYear();
      let month = String(current.getMonth() + 1).padStart(2, "0");
      let colKey = "kdec_amount_" + year + month;
      let colCaption = year + "年" + month + "月";

      // 创建报表列对象
      let col = new Object();
      col["fieldKey"] = colKey;
      col["fieldName"] = colCaption;
      col["fieldType"] = "java.math.BigDecimal";
      col["scale"] = 2;
      columns.add(col);

      // 下一个月
      current.setMonth(current.getMonth() + 1);
    }

    return columns;
  }

  query(queryParam: any, selectedObj: any): any {
    let dataSet = QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "sm_salorder",
      "customer.name as kdec_customer_name, "
        + "substr(bizdate,0,7) as kdec_month, "
        + "salorderentry.amount as kdec_amount",
      null, null
    );

    // 按客户和月份分组统计
    let resultDataSet = dataSet.copy()
      .groupBy(["kdec_customer_name", "kdec_month"])
      .sum("kdec_amount", "kdec_month_amount")
      .finish();

    return resultDataSet;
  }
}

let plugin = new MonthlyAmountReportDataPlugin();
export { plugin };
```

**要点**：
- `getColumns` 在 `query` 之前调用，可根据查询参数动态增删列
- 动态列的 `fieldKey` 需与 `query` 返回的 DataSet 字段名对应
- 常用于交叉表（行转列）场景

### 5. getQueryParam - 获取查询条件

**方法签名**：`getQueryParam(): any`

**说明**：非事件方法，通过反射赋值。插件可调用此方法获取当前报表查询参数对象。

**业务场景**：在 `getColumns` 或其他方法中获取过滤面板的查询条件

```typescript
import { AbstractReportListDataPlugin } from "@cosmic/bos-core/kd/bos/entity/report";

class DeptBudgetReportDataPlugin extends AbstractReportListDataPlugin {

  /**
   * 在动态列等场景中使用 getQueryParam 获取查询条件
   */
  getColumns(columns: any): any {
    // 通过 getQueryParam 获取当前查询参数
    let queryParam = this.getQueryParam();
    if (queryParam == null) {
      return columns;
    }

    // 获取过滤面板中的预算类型
    let filterItems = queryParam.getFilter().getFilterItems();
    for (let item of filterItems) {
      if ("kdec_budget_type" === item.getPropName()) {
        let budgetType = item.getValue() as string;

        // 根据预算类型动态添加列
        if ("monthly" === budgetType) {
          for (let i = 1; i <= 12; i++) {
            let col = new Object();
            col["fieldKey"] = "kdec_month_" + i;
            col["fieldName"] = i + "月预算";
            col["fieldType"] = "java.math.BigDecimal";
            columns.add(col);
          }
        } else if ("quarterly" === budgetType) {
          for (let i = 1; i <= 4; i++) {
            let col = new Object();
            col["fieldKey"] = "kdec_quarter_" + i;
            col["fieldName"] = "第" + i + "季度预算";
            col["fieldType"] = "java.math.BigDecimal";
            columns.add(col);
          }
        }
      }
    }

    return columns;
  }

  query(queryParam: any, selectedObj: any): any {
    // query 方法的 queryParam 与 getQueryParam() 获取到的是同一个对象
    // ... 查询逻辑
    return null;
  }
}

let plugin = new DeptBudgetReportDataPlugin();
export { plugin };
```

**要点**：
- `getQueryParam()` 由框架通过反射赋值，插件不需要自行设置
- 在 `query` 方法中可以直接使用参数 `queryParam`，效果相同
- 主要用于 `getColumns` 等不直接接收 queryParam 参数的方法中

### 7. queryBatchBy - 返回分页取数依据

**方法签名**：`queryBatchBy(queryParam: any): any`

**触发时机**：报表数据量大时，由 `ReportServiceImpl.checkLargeData` 调用，用于分批取数

**业务场景**：大数据量供应商对账报表 —— 按供应商分批取数，避免一次加载过多数据

```typescript
import { AbstractReportListDataPlugin } from "@cosmic/bos-core/kd/bos/entity/report";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class SupplierReconciliationReportPlugin extends AbstractReportListDataPlugin {

  /**
   * 返回分批取数的依据数据
   * 框架会根据返回的数据逐批调用 query
   * 每批对应一页数据
   */
  queryBatchBy(queryParam: any): any {
    let filters: QFilter[] = [];
    let filterItems = queryParam.getFilter().getFilterItems();
    for (let item of filterItems) {
      if ("kdec_startdate" === item.getPropName()) {
        filters.push(new QFilter("bizdate", QCP.ge, item.getValue()));
      } else if ("kdec_enddate" === item.getPropName()) {
        filters.push(new QFilter("bizdate", QCP.le, item.getValue()));
      }
    }

    // 返回供应商列表作为分页依据
    // 框架会按此列表逐批调用 query
    let batchDataSet = QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "ap_paybill",
      "supplier as kdec_supplier_id, supplier.name as kdec_supplier_name",
      filters.length > 0 ? filters : null,
      null
    );

    // 去重：每个供应商一批
    let distinctDataSet = batchDataSet.copy()
      .groupBy(["kdec_supplier_id", "kdec_supplier_name"])
      .finish();

    return distinctDataSet;
  }

  /**
   * 查询单批数据
   * queryParam 中包含 queryBatchBy 返回的当前批次数据
   */
  query(queryParam: any, selectedObj: any): any {
    let filters: QFilter[] = [];
    let filterItems = queryParam.getFilter().getFilterItems();
    for (let item of filterItems) {
      if ("kdec_startdate" === item.getPropName()) {
        filters.push(new QFilter("bizdate", QCP.ge, item.getValue()));
      } else if ("kdec_enddate" === item.getPropName()) {
        filters.push(new QFilter("bizdate", QCP.le, item.getValue()));
      }
    }

    // 查询当前供应商的对账数据
    let dataSet = QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "ap_paybill",
      "supplier.name as kdec_supplier_name, "
```

### 9. query（树形）- 查询树节点数据

**方法签名**：`query(queryParam: any, nodeId: string): any`

**触发时机**：报表左树加载或展开节点时调用

**业务场景**：按组织架构展示部门树 —— 点击部门节点后右侧显示该部门的费用明细

```typescript
import { AbstractReportTreeDataPlugin } from "@cosmic/bos-core/kd/bos/entity/report";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { ArrayList } from "@cosmic/bos-script/java/util";

class OrgTreeDataPlugin extends AbstractReportTreeDataPlugin {

  /**
   * 查询树节点数据
   * @param queryParam 查询参数
   * @param nodeId 当前展开的节点 ID，根节点时为 null 或空
   * @returns TreeNode 列表
   */
  query(queryParam: any, nodeId: string): any {
    let treeNodes = new ArrayList();

    if (nodeId == null || nodeId === "" || nodeId === "0") {
      // 根节点：查询所有一级组织
      let orgData = QueryServiceHelper.query(
        "bos_org",
        "id,name,number",
        [new QFilter("parentorg", QCP.equals, 0)],
        "number"
      );

      for (let org of orgData) {
        let node = new Object();
        node["id"] = org.getString("id");
        node["text"] = org.getString("name");
        node["parentId"] = "0";
        node["isLeaf"] = false;
        treeNodes.add(node);
      }
    } else {
      // 子节点：查询指定组织下的部门
      let deptData = QueryServiceHelper.query(
        "bos_org",
        "id,name,number",
        [new QFilter("parentorg", QCP.equals, nodeId)],
        "number"
      );

      // 判断是否有下级
      for (let dept of deptData) {
        let deptId = dept.getString("id");
        let hasChildren = QueryServiceHelper.exists(
          "bos_org",
          [new QFilter("parentorg", QCP.equals, deptId)]
        );

        let node = new Object();
        node["id"] = deptId;
        node["text"] = dept.getString("name");
        node["parentId"] = nodeId;
        node["isLeaf"] = !hasChildren;
        treeNodes.add(node);
      }
    }

    return treeNodes;
  }
}

let plugin = new OrgTreeDataPlugin();
export { plugin };
```

**要点**：
- `nodeId` 为 null 或空时表示加载根节点
- 每个 TreeNode 需包含 `id`、`text`、`parentId`、`isLeaf` 属性
- `isLeaf = true` 表示叶子节点，不可继续展开
- 树形数据插件与列表数据插件配合使用：选中树节点后触发右侧列表查询

### 16. processRowData - 行数据处理事件

**方法签名**：`processRowData(gridPk: string, rowData: any, queryParam: any): void`

**触发时机**：报表数据返回后、前端渲染前，由 `ReportListModel.processRowData` 调用

**业务场景**：利润分析报表 —— 计算利润率并标记亏损行

```typescript
import { AbstractReportFormPlugin } from "@cosmic/bos-core/kd/bos/report/plugin";
import { BusinessDataServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";

class ProfitReportFormPlugin extends AbstractReportFormPlugin {

  /**
   * 处理报表行数据
   * 计算利润率，并将最后一行设置为合计行
   */
  processRowData(gridPK: string, rowData: any, queryParam: any): void {
    if (rowData == null || rowData.size() == 0) {
      return;
    }

    for (let i = 0; i < rowData.size(); i++) {
      let row = rowData.get(i);

      let revenue = row.getBigDecimal("kdec_revenue");
      let cost = row.getBigDecimal("kdec_cost");

      if (revenue != null && cost != null) {
        // 计算利润
        let profit = revenue.subtract(cost);
        row.set("kdec_profit", profit);

        // 计算利润率
        if (revenue.compareTo(BigDecimal.ZERO) != 0) {
          let profitRate = profit.multiply(new BigDecimal("100"))
            .divide(revenue, 2, BigDecimal.ROUND_HALF_UP);
          row.set("kdec_profit_rate", profitRate.toString() + "%");
        } else {
          row.set("kdec_profit_rate", "0%");
        }
      }
    }

    // 将最后一行标记为合计行，设置名称为"合计"
    let lastRow = rowData.get(rowData.size() - 1);
    if (lastRow.get("kdec_customer") == null) {
      let totalObj = BusinessDataServiceHelper.newDynamicObject("bd_customer");
      totalObj.set("name", "合计");
      lastRow.set("kdec_customer", totalObj);
    }

    super.processRowData(gridPK, rowData, queryParam);
  }
}
```

### 19. filterContainerSearchClick - 过滤容器搜索点击事件

**方法签名**：`filterContainerSearchClick(args: any): void`

**触发时机**：过滤面板点击"查询"按钮时

**业务场景**：查询前校验过滤条件 —— 日期范围不能超过一年

```typescript
import { AbstractReportFormPlugin } from "@cosmic/bos-core/kd/bos/report/plugin";

class ValidateFilterReportPlugin extends AbstractReportFormPlugin {

  /**
   * 过滤面板"查询"按钮点击时校验
   */
  filterContainerSearchClick(args: any): void {
    super.filterContainerSearchClick(args);

    // 获取过滤条件
    let filterInfo = args.getFilterInfo();
    if (filterInfo == null) {
      return;
    }

    let startDate: Date = null;
    let endDate: Date = null;

    let filterItems = filterInfo.getFilterItems();
    for (let item of filterItems) {
      if ("kdec_startdate" === item.getPropName()) {
        startDate = item.getValue() as Date;
      } else if ("kdec_enddate" === item.getPropName()) {
        endDate = item.getValue() as Date;
      }
    }

    // 校验日期范围不能超过一年
    if (startDate != null && endDate != null) {
      let diffMs = endDate.getTime() - startDate.getTime();
      let diffDays = diffMs / (1000 * 60 * 60 * 24);

      if (diffDays > 366) {
        this.getView().showTipNotification("查询日期范围不能超过一年，请缩小范围");
        args.setCancel(true);
        return;
      }

      if (diffDays < 0) {
        this.getView().showTipNotification("开始日期不能晚于结束日期");
        args.setCancel(true);
        return;
      }
    }

    // 校验必选条件
    if (startDate == null || endDate == null) {
      this.getView().showTipNotification("请选择完整的日期范围");
      args.setCancel(true);
    }
  }
}
```

```typescript
   * 导出前对数据进行加工
   * @param exportColumns 导出的列信息
   * @param rowData 导出的行数据（DynamicObjectCollection）

  preProcessExportData(exportColumns: any, rowData: any): void {
    super.preProcessExportData(exportColumns, rowData);

    if (rowData == null || rowData.size() == 0) {
      return;
    }

    for (let i = 0; i < rowData.size(); i++) {
```

### 30. setExcelName - 自定义报表导出名称

**方法签名**：`setExcelName(list: any): void`

**触发时机**：导出 Excel 时获取文件名，由 `AbstractReportExport.getFileName` 调用

**业务场景**：导出文件名包含日期和查询条件 —— 如"销售报表_2024年01月_华北区域.xlsx"

```typescript
import { AbstractReportFormPlugin } from "@cosmic/bos-core/kd/bos/report/plugin";

class ExcelNamePlugin extends AbstractReportFormPlugin {

  /**
   * 自定义报表导出的 Excel 文件名
   * @param list 文件名列表（字符串列表）
   */
  setExcelName(list: any): void {
    super.setExcelName(list);

    // 清空默认名称
    list.clear();

    // 获取查询条件中的日期
    let startDate = this.getModel().getValue("kdec_startdate") as Date;
    let endDate = this.getModel().getValue("kdec_enddate") as Date;

    // 获取查询条件中的组织
    let orgObj = this.getModel().getValue("kdec_org");
    let orgName = "";
    if (orgObj != null) {
      orgName = "_" + orgObj.getString("name");
    }

    // 构造文件名
    let fileName = "销售报表";
    if (startDate != null && endDate != null) {
      let startStr = startDate.getFullYear() + "年"
        + String(startDate.getMonth() + 1).padStart(2, "0") + "月";
      let endStr = endDate.getFullYear() + "年"
        + String(endDate.getMonth() + 1).padStart(2, "0") + "月";

      if (startStr === endStr) {
        fileName += "_" + startStr;
      } else {
        fileName += "_" + startStr + "至" + endStr;
      }
    }

    fileName += orgName;

    list.add(fileName);
  }
}

let plugin = new ExcelNamePlugin();
export { plugin };
```

**要点**：
- `list` 是字符串列表，通常只需设置一个文件名
- 文件名不需要包含 `.xlsx` 后缀，框架会自动添加
- 可以结合查询条件生成有意义的文件名

### 32. resetDataCount - 重设报表总行数

**方法签名**：`resetDataCount(): any`

**触发时机**：报表数据加载后，由 `ReportList.getData` 调用

**业务场景**：报表实际行数与页面显示行数不一致时 —— 分组合并后实际显示行数少于数据行数

```typescript
import { AbstractReportFormPlugin } from "@cosmic/bos-core/kd/bos/report/plugin";

class ResetDataCountPlugin extends AbstractReportFormPlugin {

  /**
   * 重设前端显示的报表总行数
   * 当插件对数据做了合并/分组等处理后，实际显示行数可能与数据行数不同
   * @returns 重设后的总行数，返回 null 时使用默认行数
   */
  resetDataCount(): any {
    // 场景1：有合计行时，总行数减 1
    // 避免分页器将合计行计入总数
    let originalCount = 100; // 假设原始数据 100 行
    let hasTotal = true;     // 最后一行是合计行

    if (hasTotal) {
      return originalCount - 1;
    }

    // 场景2：树形报表中，只统计叶子节点行数
    // let leafCount = this.calculateLeafCount();
    // return leafCount;

    // 场景3：返回 null 使用默认行数
    return null;
  }
}

let plugin = new ResetDataCountPlugin();
export { plugin };
```

**要点**：
- 返回 `null` 时使用框架默认的行数
- 主要影响报表底部的分页器显示
- 适用于数据经过合并、分组等处理后行数变化的场景

## 附录：完整的报表插件组合示例

以下是一个完整的报表项目，包含查询插件和表单插件的配合使用。

### 业务场景

采购价格分析报表：按供应商、物料维度统计采购均价及波动，支持左树选择组织、列头过滤排序、合计行、条件样式、自定义导出名。

### 报表查询插件

```typescript
import { AbstractReportListDataPlugin } from "@cosmic/bos-core/kd/bos/entity/report";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class PurchasePriceAnalysisDataPlugin extends AbstractReportListDataPlugin {

  query(queryParam: any, selectedObj: any): any {
    let filters: QFilter[] = [];

    // 左树选中的组织
    if (selectedObj != null) {
      filters.push(new QFilter("org", QCP.equals, selectedObj.toString()));
    }

    // 过滤面板条件
    let filterItems = queryParam.getFilter().getFilterItems();
    for (let item of filterItems) {
      if ("kdec_startdate" === item.getPropName()) {
        filters.push(new QFilter("bizdate", QCP.ge, item.getValue()));
      } else if ("kdec_enddate" === item.getPropName()) {
        filters.push(new QFilter("bizdate", QCP.le, item.getValue()));
      } else if ("kdec_supplier.id" === item.getPropName()) {
        filters.push(new QFilter("supplier", QCP.in, item.getValue()));
      }
    }

    this.setProgress(0);

    // 查询采购明细
    let dataSet = QueryServiceHelper.queryDataSet(
      this.getClass().getName(),
      "pm_purorderbill",
      "supplier.name as kdec_supplier_name, "
        + "billentry.material.name as kdec_material_name, "
        + "billentry.priceandtax as kdec_price, "
        + "billentry.qty as kdec_qty, "
        + "billentry.amountandtax as kdec_amount",
      filters.length > 0 ? filters : null,
      null
    );
    this.setProgress(50);

    // 按供应商+物料分组统计
    let resultDataSet = dataSet.copy()
      .groupBy(["kdec_supplier_name", "kdec_material_name"])
      .sum("kdec_qty", "kdec_total_qty")
      .sum("kdec_amount", "kdec_total_amount")
      .avg("kdec_price", "kdec_avg_price")
      .min("kdec_price", "kdec_min_price")
      .max("kdec_price", "kdec_max_price")
      .count("kdec_order_count")
      .finish();
    this.setProgress(100);

    return resultDataSet;
  }
}

let plugin = new PurchasePriceAnalysisDataPlugin();
export { plugin };
```

### 报表表单插件

```typescript
import { AbstractReportFormPlugin } from "@cosmic/bos-core/kd/bos/report/plugin";
import { RequestContext } from "@cosmic/bos-core/kd/bos/context";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { ArrayList } from "@cosmic/bos-script/java/util";

class PurchasePriceAnalysisFormPlugin extends AbstractReportFormPlugin {

  // 初始化默认查询条件
  initDefaultQueryParam(queryParam: any): void {
    super.initDefaultQueryParam(queryParam);
    let now = new Date();
    let firstDay = new Date(now.getFullYear(), now.getMonth(), 1);
    queryParam.getFilter().addFilterItem("kdec_startdate", firstDay);
    queryParam.getFilter().addFilterItem("kdec_enddate", now);
  }

  // 过滤面板基础资料选择前过滤
  filterContainerBeforeSelect(args: any): void {
    super.filterContainerBeforeSelect(args);
    if ("kdec_supplier" === args.getFieldName()) {
      let filters = new ArrayList();
      filters.add(new QFilter("status", QCP.equals, "C"));
      args.setCustomQFilters(filters);
    }
  }

  // 查询前校验
  beforeQuery(queryParam: any): void {
    super.beforeQuery(queryParam);
    let hasDate = false;
    let filterItems = queryParam.getFilter().getFilterItems();
    for (let item of filterItems) {
      if ("kdec_startdate" === item.getPropName()) {
        hasDate = true;
      }
    }
    if (!hasDate) {
      this.getView().showTipNotification("请选择开始日期");
      queryParam.setCancel(true);
    }
  }

  // 查询完成后提示
  afterQuery(queryParam: any): void {
    super.afterQuery(queryParam);
    this.getView().showSuccessNotification("查询完成");
  }

  // 行数据处理
  processRowData(gridPK: string, rowData: any, queryParam: any): void {
    for (let i = 0; i < rowData.size(); i++) {
      let row = rowData.get(i);
      let maxPrice = row.getBigDecimal("kdec_max_price");
      let minPrice = row.getBigDecimal("kdec_min_price");
      if (maxPrice != null && minPrice != null) {
        let priceRange = maxPrice.subtract(minPrice);
        row.set("kdec_price_range", priceRange);
      }
    }
    super.processRowData(gridPK, rowData, queryParam);
  }

  // 设置排序和过滤
  setSortAndFilter(allColumns: any): void {
    super.setSortAndFilter(allColumns);
    for (let i = 0; i < allColumns.size(); i++) {
      let col = allColumns.get(i);
      col.setSortable(true);
      col.setFilterable(true);
    }
  }

  // 设置合计行
  setFloatBottomData(summaryEvents: any): void {
    super.setFloatBottomData(summaryEvents);
    for (let i = 0; i < summaryEvents.size(); i++) {
      let event = summaryEvents.get(i);
      if ("kdec_total_amount" === event.getFieldKey()) {
        event.setSummaryType("sum");
      } else if ("kdec_total_qty" === event.getFieldKey()) {
        event.setSummaryType("sum");
      } else if ("kdec_avg_price" === event.getFieldKey()) {
        event.setSummaryType("avg");
      }
    }
  }

  // 条件样式

    super.setCellStyleRules(cellStyleRules);
    // 价格波动大于 50 时标红
    let rule = new Object();
    rule["fieldKey"] = "kdec_price_range";
    rule["backgroundColor"] = "#FF4444";
    rule["fontColor"] = "#FFFFFF";
    rule["condition"] = "kdec_price_range > 50";
    cellStyleRules.add(rule);
  }

    super.setMergeColumns(columns);
    columns.add("kdec_supplier_name");
  }

    list.clear();
    let now = new Date();
    let dateStr = now.getFullYear() + String(now.getMonth() + 1).padStart(2, "0");
    list.add("采购价格分析报表_" + dateStr);
  }
}

let plugin = new PurchasePriceAnalysisFormPlugin();
export { plugin };
```

## 注意事项

1. **报表查询插件和表单插件成对注册**：取数插件注册在报表列表节点，表单插件注册在页面根节点
2. **报表数据只读**：使用 `QueryServiceHelper` 而非 `BusinessDataServiceHelper`
3. **大数据量使用 DataSet**：通过 `QueryServiceHelper.queryDataSet` 获取 DataSet，利用 `groupBy`、`join`、`union` 进行数据加工
4. **无状态设计**：插件类中不要定义类属性，所有数据使用局部变量
5. **遵守 SDK 限流**：单次查询不超过 50,000 行，查询字段不超过 100 个
6. **条件样式的 condition**：使用 JS 表达式语法，可引用报表其他列字段标识
7. **进度条**：耗时查询中使用 `setProgress` 给用户反馈
