# 弹性域场景拆分

这个目录聚焦弹性域的取值、显示和值结构判断，优先回答“怎么拿到 flex 值”“怎么补显示文本”“怎么区分显示值和实际值”。

## 场景目录

- [flexValueRead.md](flexValueRead.md)
- [flexDisplayText.md](flexDisplayText.md)
- [listFlexPack.md](listFlexPack.md)

## 使用建议

- 页面录入看 `FlexEdit`，结构判断看 `FlexProp` / `FlexEntityType`。
- 先确认你要的是“值对象”还是“显示文本”，再决定走哪种写法。
