## beforeExecClientCmd - 前端指令发送前

**所属接口**：`IMobFormPlugin`
**基类**：`AbstractMobFormPlugin`
**签名**：`beforeExecClientCmd(map: Map<String, Object>, operateKey: string): void`
**触发时机**：移动端前端指令发送到服务端执行前。
**典型用途**：拦截或修改前端指令参数，如在提交前追加 GPS 坐标、设备信息等移动端特有数据。

### 业务场景：外勤拜访单提交前自动注入设备信息和网络状态

销售人员在移动端填写客户拜访单并提交，提交指令发送前自动将设备型号、网络类型注入到指令参数中，便于后台记录外勤轨迹和离线提交标识。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";

class VisitMobFormPlugin extends AbstractMobFormPlugin {

  beforeExecClientCmd(map: any, operateKey: string): void {
    super.beforeExecClientCmd(map, operateKey);

    if (operateKey === "submit") {
      // 从前端指令参数中读取设备信息（由移动端框架注入）
      let deviceModel = map.get("deviceModel") as string;
      let networkType = map.get("networkType") as string;

      // 将设备信息写入单据字段
      if (deviceModel != null) {
        this.getModel().setValue("kdec_device_model", deviceModel);
      }
      if (networkType != null) {
        this.getModel().setValue("kdec_network_type", networkType);
      }

      // 离线模式提交标记
      let isOffline = map.get("isOffline") as boolean;
      if (isOffline) {
        this.getModel().setValue("kdec_submit_source", "offline");
        this.getView().showTipNotification("离线数据将在联网后自动同步");
      } else {
        this.getModel().setValue("kdec_submit_source", "online");
      }
    }

    // 拦截特定指令：禁止移动端执行删除
    if (operateKey === "delete") {
      map.put("cancel", true);
      this.getView().showErrorNotification("移动端不允许删除拜访单，请在PC端操作");
    }
  }
}

let plugin = new VisitMobFormPlugin();
export { plugin };
```

**要点说明**：

1. `map` 包含前端指令的参数键值对，可读取也可追加/修改
2. `operateKey` 标识当前执行的操作类型（如 `submit`、`save`、`delete`）
3. 可通过向 `map` 中写入 `cancel: true` 来阻止指令继续执行
4. 移动端特有参数（设备、网络、GPS）由框架注入到 `map` 中

## locate - 移动端定位成功

**所属接口**：`IMobFormPlugin`
**基类**：`AbstractMobFormPlugin`
**签名**：`locate(e: LocateEvent): void`
**触发时机**：移动端调用定位功能成功后回调。
**典型用途**：获取 GPS 经纬度坐标，回填到表单字段，用于外勤打卡、拜访签到、配送签收等场景。

### 业务场景：外勤签到自动记录位置并校验是否在客户范围内

销售人员到达客户现场后点击"签到"，移动端定位成功后获取经纬度，与客户预设地址做距离比对，超出 500 米范围则提示异常。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { DynamicObject } from "@cosmic/bos-core/kd/bos/dataentity/entity";

class CheckInMobFormPlugin extends AbstractMobFormPlugin {

  locate(e: any): void {
    let longitude = e.getLongitude();
    let latitude = e.getLatitude();
    let address = e.getAddress();

    // 回填定位信息到表单
    this.getModel().setValue("kdec_longitude", longitude);
    this.getModel().setValue("kdec_latitude", latitude);
    this.getModel().setValue("kdec_checkin_address", address || "");
    this.getModel().setValue("kdec_checkin_time", new Date());

    // 获取当前客户的预设坐标
    let customerObj = this.getModel().getValue("kdec_customer") as DynamicObject;
    if (customerObj != null) {
      let custLocation = QueryServiceHelper.queryOne(
        "bd_customer",
        "kdec_cust_longitude,kdec_cust_latitude,name",
        [new QFilter("id", QCP.equals, customerObj.getLong("id"))]
      );

      if (custLocation != null) {
        let custLng = custLocation.getDouble("kdec_cust_longitude");
        let custLat = custLocation.getDouble("kdec_cust_latitude");

        // 计算与客户预设地址的距离（简化球面距离公式）
        let distance = this.calcDistance(latitude, longitude, custLat, custLng);

        this.getModel().setValue("kdec_distance", distance);

        if (distance > 500) {
          this.getView().showErrorNotification(
            "当前位置距客户 " + custLocation.getString("name")
            + " 约 " + Math.round(distance) + " 米，超出500米签到范围"
          );
          this.getModel().setValue("kdec_checkin_status", "abnormal");
        } else {
          this.getView().showSuccessNotification("签到成功，距客户 " + Math.round(distance) + " 米");
          this.getModel().setValue("kdec_checkin_status", "normal");
        }
      }
    }
  }

  // 简化距离计算（单位：米）
  calcDistance(lat1: number, lng1: number, lat2: number, lng2: number): number {
    let radLat1 = lat1 * Math.PI / 180;
    let radLat2 = lat2 * Math.PI / 180;
    let deltaLat = radLat1 - radLat2;
    let deltaLng = (lng1 - lng2) * Math.PI / 180;
    let s = 2 * Math.asin(Math.sqrt(
      Math.pow(Math.sin(deltaLat / 2), 2)
      + Math.cos(radLat1) * Math.cos(radLat2) * Math.pow(Math.sin(deltaLng / 2), 2)
    ));
    return s * 6371000;
  }
}

let plugin = new CheckInMobFormPlugin();
export { plugin };
```

**要点说明**：

1. `e.getLongitude()` / `e.getLatitude()` 获取 GPS 经纬度
2. `e.getAddress()` 获取逆地理编码后的地址文本，可能为空
3. 定位需要用户授权，拒绝授权时不会触发此事件
4. 距离计算使用 Haversine 公式，实际项目可调用地图 API 获取更精准结果

## refreshData - 卡片分录下拉刷新

**所属接口**：`IMobFormPlugin`
**基类**：`AbstractMobFormPlugin`
**签名**：`refreshData(): void`
**触发时机**：移动端单据体卡片分录下拉触发刷新时回调。
**典型用途**：重新查询数据并刷新分录列表，实现移动端的下拉刷新体验。

### 业务场景：移动端巡检单下拉刷新最新巡检项

巡检人员在移动端查看巡检单，下拉刷新时重新查询待检项目，将新增的巡检点追加到分录中。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";
import { DynamicObject } from "@cosmic/bos-core/kd/bos/dataentity/entity";

class InspectMobFormPlugin extends AbstractMobFormPlugin {

  refreshData(): void {
    super.refreshData();

    // 获取当前巡检路线
    let routeObj = this.getModel().getValue("kdec_inspect_route") as DynamicObject;
    if (routeObj == null) {
      this.getView().showTipNotification("请先选择巡检路线");
      return;
    }

    // 查询该路线下所有巡检点
    let checkpoints = QueryServiceHelper.query(
      "kdec_inspect_checkpoint",
      "id,name,kdec_location,kdec_check_items",
      [
        new QFilter("kdec_route.id", QCP.equals, routeObj.getLong("id")),
        new QFilter("enable", QCP.equals, "1")
      ],
      "kdec_seq"
    );

    // 获取已有分录中的巡检点 ID 集合
    let existingIds = new ArrayList();
    let rowCount = this.getModel().getEntryRowCount("kdec_entryentity");
    for (let i = 0; i < rowCount; i++) {
      let cpObj = this.getModel().getValue("kdec_checkpoint", i) as DynamicObject;
      if (cpObj != null) {
        existingIds.add(cpObj.getLong("id"));
      }
    }

    // 追加新增的巡检点
    let addedCount = 0;
    for (let cp of checkpoints) {
      let cpId = cp.getLong("id");
      if (!existingIds.contains(cpId)) {
        let newIndex = this.getModel().createNewEntryRow("kdec_entryentity");
        this.getModel().setValue("kdec_checkpoint", cpId, newIndex);
        this.getModel().setValue("kdec_check_status", "pending", newIndex);
        addedCount++;
      }
    }

    if (addedCount > 0) {
      this.getView().showSuccessNotification("刷新完成，新增 " + addedCount + " 个巡检点");
    } else {
      this.getView().showTipNotification("已是最新数据");
    }
  }
}

let plugin = new InspectMobFormPlugin();
export { plugin };
```

**要点说明**：

1. `refreshData` 由移动端下拉手势自动触发
2. 刷新时应做增量处理，避免清空已有数据重新加载
3. 刷新完成后需给用户明确的反馈提示
4. 注意 SDK 限流：单次查询不超过 50,000 行

## timeZoneLocate - 移动端时区设置

**所属接口**：`IMobFormPlugin`
**基类**：`AbstractMobFormPlugin`
**签名**：`timeZoneLocate(e: TimeZoneLocationEvent): void`
**触发时机**：移动端获取到设备时区信息后回调。
**典型用途**：根据设备所在时区调整单据的业务日期和时间显示，适用于跨时区业务（跨国采购、海外仓库管理等）。

### 业务场景：跨国采购订单根据当地时区自动换算业务时间

采购人员在海外出差时通过移动端录入采购订单，系统根据设备时区自动将业务日期转换为总部时区（北京时间），并在单据上同时显示当地时间和总部时间。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";

class PurOrderMobFormPlugin extends AbstractMobFormPlugin {

  timeZoneLocate(e: any): void {
    super.timeZoneLocate(e);

    let timeZoneId = e.getTimeZoneId();       // 如 "America/New_York"
    let timeZoneOffset = e.getTimeZoneOffset(); // 如 -5（相对 UTC 偏移小时数）

    // 记录设备时区
    this.getModel().setValue("kdec_local_timezone", timeZoneId);
    this.getModel().setValue("kdec_timezone_offset", timeZoneOffset);

    // 计算总部时间（北京时间 = UTC+8）
    let localNow = new Date();
    let utcTimestamp = localNow.getTime() + (timeZoneOffset * 60 * 60 * 1000);
    let beijingOffset = 8 * 60 * 60 * 1000;
    let beijingTime = new Date(utcTimestamp + beijingOffset);

    // 回填总部时间为业务日期
    this.getModel().setValue("kdec_bizdate", beijingTime);
    this.getModel().setValue("kdec_local_time", localNow);

    // 时差提示
    let diffHours = 8 - timeZoneOffset;
    if (diffHours !== 0) {
      this.getView().showTipNotification(
        "当前时区：" + timeZoneId + "，与总部时差 " + diffHours + " 小时"
      );
    }
  }
}

let plugin = new PurOrderMobFormPlugin();
export { plugin };
```

**要点说明**：

1. `e.getTimeZoneId()` 返回标准时区 ID（如 `Asia/Shanghai`、`America/New_York`）
2. `e.getTimeZoneOffset()` 返回相对 UTC 的偏移小时数
3. 跨时区场景下，业务日期统一用总部时区，避免数据混乱
4. 此事件仅在移动端触发，PC 端不触发

## uploadFile - 文件上传成功

**所属接口**：`IMobFormPlugin`
**基类**：`AbstractMobFormPlugin`
**签名**：`uploadFile(e: EventObject): void`
**触发时机**：移动端拍照上传或文件上传成功后回调。
**典型用途**：上传完成后获取文件信息，校验文件类型和大小，将文件信息关联到单据字段。

### 业务场景：质检报告拍照上传后自动 OCR 识别并填写检验结果

质检人员在移动端对产品质检报告拍照上传，上传成功后获取图片信息，记录附件数量，并提示用户进行后续检验操作。

```typescript
import { AbstractMobFormPlugin } from "@cosmic/bos-core/kd/bos/form/plugin";
import { QueryServiceHelper } from "@cosmic/bos-core/kd/bos/servicehelper";
import { QFilter, QCP } from "@cosmic/bos-core/kd/bos/orm/query";

class QcReportMobFormPlugin extends AbstractMobFormPlugin {

  uploadFile(e: any): void {
    let fileInfo = e.getSource();
    let fileName = fileInfo.getString("fileName");
    let fileSize = fileInfo.getLong("fileSize");
    let fileUrl = fileInfo.getString("url");
    let fileType = fileInfo.getString("fileType");

    // ---- 1. 文件校验 ----
    // 只允许图片类型
    let allowedTypes = ["image/jpeg", "image/png", "image/bmp"];
    let isAllowed = false;
    for (let i = 0; i < allowedTypes.length; i++) {
      if (allowedTypes[i] === fileType) {
        isAllowed = true;
        break;
      }
    }

    if (!isAllowed) {
      this.getView().showErrorNotification("只支持 JPG/PNG/BMP 格式的图片，当前：" + fileType);
      return;
    }

    // 文件大小限制 10MB
    let maxSize = 10 * 1024 * 1024;
    if (fileSize > maxSize) {
      this.getView().showErrorNotification("文件大小超过10MB限制");
      return;
    }

    // ---- 2. 记录上传信息 ----
    this.getModel().setValue("kdec_photo_url", fileUrl);
    this.getModel().setValue("kdec_photo_name", fileName);
    this.getModel().setValue("kdec_upload_time", new Date());

    // 更新已上传附件计数
    let currentCount = this.getModel().getValue("kdec_attachment_count") as number;
    this.getModel().setValue("kdec_attachment_count", (currentCount || 0) + 1);

    // ---- 3. 检查是否满足最低附件要求 ----
    let materialObj = this.getModel().getValue("kdec_material") as any;
    if (materialObj != null) {
      let requirement = QueryServiceHelper.queryOne(
        "kdec_qc_standard",
        "kdec_min_photos",
        [new QFilter("kdec_material.id", QCP.equals, materialObj.getLong("id"))]
      );

      if (requirement != null) {
        let minPhotos = requirement.getInt("kdec_min_photos");
        let uploaded = (currentCount || 0) + 1;
        if (uploaded < minPhotos) {
          this.getView().showTipNotification(
            "已上传 " + uploaded + " 张，该物料至少需要 " + minPhotos + " 张质检照片"
          );
        } else {
          this.getView().showSuccessNotification("照片数量已满足质检要求");
        }
      }
    }

    this.getView().showSuccessNotification("文件 " + fileName + " 上传成功");
  }
}

let plugin = new QcReportMobFormPlugin();
export { plugin };
```

**要点说明**：

1. `e.getSource()` 返回上传文件的信息对象，包含 `fileName`、`fileSize`、`url`、`fileType`
2. 上传成功后文件已存储在服务端，`url` 为可访问的文件路径
3. 应在回调中校验文件类型和大小，非法文件及时提示
4. 此事件同时适用于拍照上传和文件选择上传两种方式
