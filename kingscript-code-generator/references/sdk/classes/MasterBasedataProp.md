# MasterBasedataProp

## 基本信息

- 名称: `MasterBasedataProp`
- Java 类名: `kd.bos.entity.property.MasterBasedataProp`
- 模块: `@cosmic/bos-core`
- 所属包: `kd/bos/entity/property`
- 类型: 主数据基础资料字段元数据

## 用途概览

`MasterBasedataProp` 适合描述带主数据、主档、多组织资料语义的基础资料字段。它通常出现在多组织、多主档或分配控制更复杂的资料场景里。

## 什么时候优先看它

- 字段引用的是主数据而不是普通基础资料
- 问题涉及主数据控制、主档组织、资料分配
- 同一个基础资料字段在不同组织下表现不同
