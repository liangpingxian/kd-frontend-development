# FlexEntityType

## 基本信息

- 名称: `FlexEntityType`
- Java 类名: `kd.bos.entity.FlexEntityType`
- 模块: `@cosmic/bos-core`
- 所属包: `kd/bos/entity`
- 类型: 弹性域实体类型

## 用途概览

`FlexEntityType` 描述弹性域组合对象的实体结构。它解决的是“弹性域里有哪些维度、每个维度是什么字段、底层结构长什么样”。

## 典型场景

- 判断弹性域包含哪些维度
- 通用导出或通用显示时拆解弹性域
- 排查弹性域保存结构和显示结构不一致
