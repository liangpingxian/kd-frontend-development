## 用途概述

用于承载单据列表类控件的能力，常见于列表展示、列表扩展、移动控件场景。

## 典型场景

- 列表控件扩展
- 列表项渲染、附加项处理
- 单据列表界面的交互增强

## 用户常见问法

- 列表控件怎么扩展
- `BillList` 能干什么
- 列表里怎么加附加行为

## 常见搭配

- 列表插件
- 控件扩展插件
- `ListView`

## 高价值规则

- `BillList` 更偏控件层，不是 MVC 视图层本身
- 如果问题偏“列表页面整体行为”，优先看 `ListView`
- 如果问题偏“某个列表控件怎么扩展”，再看 `BillList`
