## 用途概述

用于承载报表列表展示能力，是报表数据结果展示层的核心对象之一。

## 典型场景

- 报表结果展示
- 报表列表列控制
- 报表导出前后的列表处理

## 高价值规则

- 报表列表偏“结果展示层”
- 如果问题偏“数据怎么查”，要看报表数据插件
- 如果问题偏“结果如何展现、导出、列怎么变”，`ReportList` 更相关
