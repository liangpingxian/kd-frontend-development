# FlexProperty

## 基本信息

- 名称: `FlexProperty`
- Java 类名: `kd.bos.entity.flex.FlexProperty`
- 模块: `@cosmic/bos-core`
- 所属包: `kd/bos/entity/flex`
- 类型: 弹性域维度属性

## 用途概览

`FlexProperty` 更偏弹性域内部维度级定义。它适合回答“某个弹性域里的某一段维度是什么、怎么显示、怎么取值”这类问题。

## 典型场景

- 弹性域维度遍历
- 弹性域显示值拼接
- 动态判断某个 flex 维度是否存在
