# FlexProp

## 基本信息

- 名称: `FlexProp`
- Java 类名: `kd.bos.entity.property.FlexProp`
- 模块: `@cosmic/bos-core`
- 所属包: `kd/bos/entity/property`
- 类型: 弹性域字段元数据

## 用途概览

`FlexProp` 描述的是弹性域字段本身的元数据。它通常用来连接“表单上的弹性域控件”和“底层的弹性域实体结构”。

## 高频问法

- 这个字段是不是弹性域
- 弹性域字段和普通字符串字段怎么区分
- 弹性域取到的对象结构该怎么看
