# DBRoute

## 基本信息

- 名称: `DBRoute`
- Java 类名: `kd.bos.db.DBRoute`
- 模块: `@cosmic/bos-core`
- 所属包: `kd/bos/db`
- 类型: 数据库路由对象

## 用途概览

`DBRoute` 用来描述数据库访问路由。凡是直接使用数据库能力、分库或按路由取主键的场景，都可能碰到它。

## 高频用法

| 方法 | 作用 |
|------|------|
| `of(...)` | 创建路由 |
| `getPrimaryKeys(...)` | 在指定路由上取主键 |

## 运行时注意事项

- 直接数据库访问属于高风险能力，优先确认有没有更高层的 service/helper 可用。
- 路由错了通常不是“查不到”这么简单，还可能带来组织边界或数据隔离问题。
