# BasedataProp

## 基本信息

- 名称: `BasedataProp`
- Java 类名: `kd.bos.entity.property.BasedataProp`
- 模块: `@cosmic/bos-core`
- 所属包: `kd/bos/entity/property`
- 类型: 基础资料字段元数据

## 用途概览

`BasedataProp` 描述的是基础资料字段本身的元数据，比如引用的资料类型、字段结构、显示规则等。它是“基础资料字段长什么样”的入口，不是“当前控件怎么弹 F7”的入口。

## 典型场景

- 判断某字段是不是基础资料字段
- 通用工具里识别字段引用类型
- 排查基础资料字段回填异常时确认元数据定义
