## 如何使用

- 需要确认某个关键字能不能用时，先看 [保留关键字.md](保留关键字.md)
- 需要确认代码结构和书写约定时，先看 [命名规范.md](命名规范.md)
- 需要确认基本语法时，按上面的章节顺序阅读
- 需要快速看一组可直接模仿的写法时，先看 [语法示例.md](语法示例.md)
- 需要定位 `import`、模块边界和引用方式时，直接看 [模块及引用.md](模块及引用.md)

## 使用边界

- 这些文档描述的是 Kingscript 语言层面的约束，不代表运行时 API 一定可用
- 语言语法正确，不等于平台权限、租户隔离和生命周期时序正确
- 语法问题与 SDK 问题要分开判断；涉及 SDK 时再配合 `../../sdk/` 中的索引和知识卡一起看
