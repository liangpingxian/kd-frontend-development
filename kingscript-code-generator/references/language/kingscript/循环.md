## 循环

在 KingScript 中，循环用于重复执行代码块，主要包括 `for` 循环和 `while` 循环。

### 1.1 for 循环

`for` 循环适用于已知循环次数的情况。

```kingscript
for (let i = 0; i < 5; i++) {
    console.log("当前值：" + i);
}
```

### 1.2 遍历对象数组

如果有多个对象组成的数组，并需要筛选出符合特定条件的对象，可以使用 `for...of` 进行遍历：

```kingscript
let users = [
    { name: "Alice", age: 30 },
    { name: "Bob", age: 40 },
    { name: "Charlie", age: 25 }
];

for (let user of users) {
    if (user.age > 30) {
        console.log(user.name + " 年龄超过 30 岁");
    }
}
```

### 1.3 while 循环

`while` 循环适用于循环次数未知但依赖于某个条件的情况。

```kingscript
let count = 0;
while (count < 5) {
    console.log("当前值：" + count);
    count++;
}
```

### 1.4 退出循环

可以使用 `break` 退出循环，

```kingscript
for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break;  // 退出循环
    }
    console.log(i);
}
```
或使用 `continue` 跳过当前迭代。

```kingscript
for (let i = 0; i < 10; i++) {
    if (i % 2 === 0) {
        continue;  // 跳过偶数
    }
    console.log(i);
}
```
