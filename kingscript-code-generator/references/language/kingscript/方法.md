## 方法

在KingScript中，方法允许我们封装可复用的代码逻辑，并通过参数和返回值来进行灵活的操作。

### 1. 定义方法

方法由`function`关键字声明，后面跟方法名和参数列表。

#### 1.1 普通方法

```kingscript
function greet(name: string): string {
    return `Hello, ${name}!`;
}

console.log(greet("KingScript"));  // 输出：Hello, KingScript!
```

#### 1.2 默认参数
KingScript允许为方法的参数指定默认值。如果调用方法时未提供该参数，默认值将被使用。

```kingscript
function greet(name: string = "Guest"): string {
    return `Hello, ${name}!`;
}

console.log(greet());       // 输出：Hello, Guest!
console.log(greet("Kingdee"));  // 输出：Hello, Kingdee!
```

#### 1.3 可变参数
方法也可以接收一个不定数量的参数，这些参数会被封装成一个数组。
```kingscript
function sum(...numbers: number[]): number {
    return numbers.reduce((acc, num) => acc + num, 0);
}

console.log(sum(1, 2, 3, 4));  // 输出：10
```

### 2. 方法重载
在KingScript中，我们也可以定义参数允许多个类型。
```kingscript
//重载一
function padLeft(padding: number, input: string): string {
    return " ".repeat(padding) + input;
}
//重载二
function padLeft(padding: string, input: string): string {
    return padding + input;
}
//上述两个重载方法可以合并为一个方法
function padLeft(padding: number | string, input: string): string {
    if (typeof padding === "number") {
      return " ".repeat(padding) + input;
    }
    return padding + input;
}
```

### 3. 箭头函数
KingScript支持箭头函数的语法，这使得定义匿名方法变得更加简洁。
```kingscript
const multiply = (a: number, b: number): number => a * b;

console.log(multiply(5, 3));  // 输出：15
```

### 4. 结论
方法是实现代码复用和逻辑封装的关键组成部分。通过掌握方法的定义、重载、箭头函数等特性，
可以有效地组织代码，提升开发效率。
