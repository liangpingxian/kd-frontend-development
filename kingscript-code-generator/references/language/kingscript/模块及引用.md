## 模块和引用

苍穹平台使用了KingScript对其Java SDK API进行了封装，封装后的SDK称为Script SDK API，API按功能类型分组成包，称为脚本模块(类似npm模块)。

### 1 常用脚本模块
- @cosmic/bos-script: 脚本引擎模块，定义了基本类型和全局变量。
- @cosmic/bos-util: 公共工具模块，提供json、http等。
- @cosmic/bos-core: 苍穹平台核心功能模块，包含id、db、orm、mq、algo、log等。
- @cosmic/bos-framework: 应用框架模块，包括插件、控件、事件等。

另外SDK还有内部模块，模块名含-internal-字样，仅平台使用，二次开发勿用，如@cosmic/bos-internal-jdk、@cosmic/bos-internal-sdk，被限制执行，会抛中断异常。

### 2 模块命名规范
金蝶云苍穹·脚本开发平台会自动生成脚本路径，生成规则如下：
```
@产品/云-ISV/应用/脚本插件编码.ts
```
开发者可根据该路径在代码资源文件中快速定位脚本位置。

### 3 引入模块
我们可以在脚本中引入JAVA常见的类型，例如 ArrayList，但注意目前导入的类型不支持泛型。

```kingscript
import { ArrayList } from '@cosmic/bos-script/java/util'

let list = new ArrayList();
list.add("hello");
```
