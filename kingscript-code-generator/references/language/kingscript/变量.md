## 变量

在 KingScript 中，变量用于存储数据，并可以在程序的不同部分被访问和修改。KingScript 继承自 TypeScript，因此变量的声明方式与 TypeScript 类似。

### 1.1 变量声明

在 KingScript 中，变量可以使用 `let` 或 `const` 关键字来声明：

```kingscript
let message: string = "Hello, KingScript!";
const amount = new BigDecimal("3.33");  // 在财务计算时请始终使用 BigDecimal，避免使用 number 类型以防止浮点运算误差
```

- `let` 用于声明可以更改的变量。
- `const` 用于声明不可更改（常量）的变量。

### 1.2 变量类型

KingScript 支持多种数据类型，包括：

- `string`（字符串）：文本数据
- `number`（数字）：整数和浮点数，在财务计算时请始终使用 BigDecimal
- `boolean`（布尔值）：`true` 或 `false`
- `array`（数组）：一组相同类型的值
- `object`（对象）：键值对集合

示例：

```kingscript
let username: string = "Alice";
let age: number = 30;
let isStudent: boolean = false;
let numbers: number[] = [1, 2, 3, 4, 5];
let user: { name: string; age: number } = { name: "Alice", age: 30 };
```

### 1.3 变量的初始化和修改

使用 `let` 声明的变量可以被重新赋值，而 `const` 声明的变量则不能被修改。

```kingscript
let greeting: string = "Hello";
greeting = "Hi there!";  // 允许

const country: string = "China";
country = "USA";  // 错误，const 变量不可修改
```

### 1.4 Hello, World!

我们可以使用变量来存储 `Hello, World!` 这一经典文本，并将其输出到控制台：

```kingscript
let message: string = "Hello, World!";
console.log(message);
```

输出结果：

```
Hello, World!
```
