## 说明
`KingScript` 基于 `TypeScript` 语法，具体保留关键字如下：

### 1. JavaScript 保留关键字

| 保留关键字     | 说明 |
|-----------|-----|
| `break`     |	用于终止循环或 `switch` 语句。|
| `case`      |		用于 `switch` 语句中，指定一个条件分支。|
| `catch`     |		用于 `try...catch` 语句中，捕捉异常。|
| `class`     |		用于定义类。（ES6 引入）|
| `const`     |		用于声明常量，值一旦赋值后不可更改。|
| `continue`  |		用于跳过当前循环的剩余部分，继续下一次循环。|
| `debugger`  |		用于在代码中设置断点，调试时暂停执行。|
| `default`   |		用于 `switch` 语句中，指定默认分支。|
| `delete`    |		用于删除对象的属性。|
| `do`        |		用于 `do...while` 循环，先执行循环体再判断条件。|
| `else`      |		用于 `if...else` 语句中，指定条件不满足时执行的代码块。|
| `enum`      |		用于定义枚举类型。（ES6 引入）|
| `export`    |		用于导出模块、函数、类等，使其在其他模块中可用。（ES6 引入）|
| `extends`   |		用于类声明中，指定继承的父类。（ES6 引入）|
| `false`     |		布尔值，表示假。|
| `finally`   |		用于 `try...finally` 语句中，指定无论是否发生异常都要执行的代码块。|
| `for`       |		用于 `for` 循环，重复执行代码块。|
| `function`  |		用于声明函数。|
| `if`        |		用于条件判断，根据条件执行不同的代码块。|
| `implements` |		用于类声明中，指定类实现的接口。|
| `import`    |		用于导入模块、函数、类等。（ES6 引入）|
| `in`        |		用于 `for...in` 循环，遍历对象的可枚举属性。|
| `instanceof` |		用于判断对象是否是某个类的实例。|
| `interface` |		用于定义接口，声明对象的结构。|
| `let`       |		用于声明块级作用域的变量。（ES6 引入）|
| `new`       |		用于创建对象的实例。|
| `null`      |		表示空值。|
| `package`   |		用于声明模块的包信息。|
| `private`   |		用于类中，声明私有成员。|
| `protected` |		用于类中，声明受保护成员。|
| `public`    |		用于类中，声明公共成员。|
| `return`    |		用于从函数中返回值。|
| `static`    |		用于类中，声明静态成员。|
| `super`     |		用于访问父类的构造函数和方法。|
| `switch`    |		用于 `switch` 语句，进行多重条件判断。|
| `this`      |		用于引用当前对象的实例。|
| `throw`     |		用于抛出异常。|
| `true`      |		布尔值，表示真。|
| `try`       |		用于 `try...catch` 语句中，尝试执行代码块。|
| `typeof`    |		用于获取变量的类型。|
| `var`       |		用于声明变量。（ES5 及之前版本）|
| `void`      |		用于表示函数没有返回值。|
| `while`     |		用于 `while` 循环，根据条件重复执行代码块。|
| `with`      |		用于扩展作用域链。（不推荐使用）|
| `yield`     |		用于生成器函数中，暂停和恢复生成器。（ES6 引入）|

### 2. TypeScript 特有的保留关键字
| 保留关键字       | 说明 |
|-------------|----|
| `any`         |  表示任意类型。 |
| `boolean`     |  表示布尔类型。 |
| `constructor` |  用于在类中定义构造函数。 |
| `declare`     | 用于声明全局变量、函数等。 |
| `enum`        |  用于定义枚举类型。 |
| `number`      |  表示数值类型。 |
| `string`      |  表示字符串类型。 |
| `symbol`      | 表示符号类型。（ES6引入） |
| `type`        |  用于定义类型别名。 |
| `undefined`   |  表示未定义类型。 |

### 3. 上下文关键字

| 保留关键字     | 说明                 |
|-----------|--------------------|
| `abstract`  | 用于声明抽象类和抽象方法。      |
| `as`        | 用于类型断言，将一个值断言为特定类型。 |
| `async`     | 用于声明异步函数。          |
| `await`     | 用于等待一个 `Promise` 对象的解决。 |
| `from`      | 用于 `import` 语句中， 指定模块的路径。 |
| `get`       | 用于定义 `getter` 方法。    |
| `is`        | 用于类型谓词，声明一个函数返回的类型。 |
| `keyof`     | 用于获取一个类型的键的联合类型。   |
| `module`    | 用于声明模块。（在 ES6 中被 `namespace` 取代） |
| `namespace` | 用于声明命名空间。          |
| `never`     | 表示永远不会出现的类型。       |
| `readonly`  | 用于声明只读属性。          |
| `require`   | 用于导入模块。（在 ES6 中被 `import` 取代） |
| `set`       | 用于定义 `setter` 方法。    |
| `unique`    | 用于声明唯一类型。          |
| `unknown`   | 表示未知类型。            |
| `using`     | 用于声明使用资源。          |

| `long`         | 长整数类型。    |

| `short`        | 短整数类型。    |
| `synchronized` | 同步方法。     |
| `throws`       | 声明抛出异常。   |
| `transient`    | 瞬态变量。     |
| `volatile`     | 易变变量。     |
