## 条件判断

在 KingScript 中，我们可以使用 `if`、`else if` 和 `else` 关键字来执行条件判断，以控制代码的执行流程。

### 1.1 if 语句

`if` 语句用于在条件成立时执行代码块。

```kingscript
let score = 85;
if (score >= 60) {
    console.log("合格");
}
```

### 1.2 if-else 语句

当条件不满足时，我们可以使用 `else` 提供备用的执行逻辑。

```kingscript
let score = 50;
if (score >= 60) {
    console.log("合格");
} else {
    console.log("不合格");
}
```

### 1.3 if-else if-else 语句

如果有多个条件需要判断，可以使用 `else if`：

```kingscript
let score = 85;
if (score >= 90) {
    console.log("优秀");
} else if (score >= 60) {
    console.log("合格");
} else {
    console.log("不合格");
}
```

### 1.4 三元运算符

KingScript 也支持三元运算符（`? :`），用于简化简单的条件判断。

```kingscript
let score = 75;
let result = score >= 60 ? "合格" : "不合格";
console.log(result);
```

输出结果：

```
合格
```

### 1.5 switch 语句

`switch` 语句用于根据不同的值执行不同的代码块，适用于多个条件判断的情况。

```kingscript
let grade = "B";
switch (grade) {
    case "A":
        console.log("优秀");
        break;
    case "B":
        console.log("良好");
        break;
    case "C":
        console.log("合格");
        break;
    default:
        console.log("不及格");
}
```

输出结果（当 `grade` 为 "B" 时）：

```
良好
```

`switch` 语句相比 `if-else` 语句在多个值判断时更清晰，并且避免了冗长的 `if` 结构。
