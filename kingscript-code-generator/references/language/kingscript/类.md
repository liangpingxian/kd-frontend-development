## 类

在 KingScript 中，类是一种用于定义对象蓝图的数据结构。它允许我们创建具有相同属性和方法的对象实例。KingScript 继承自 TypeScript，因此类的语法与 TypeScript 类似。

### 1.1 类的声明

使用 `class` 关键字可以声明一个类：

```kingscript
class Person {
    // 类体
}
```

### 1.2 类的属性和方法
类可以包含属性（变量）和方法（函数）。它们可以是实例成员
```kingscript
class Person {
    name: string;
    age: number;

    constructor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }

    greet(): void {
        console.log(`Hello, my name is ${this.name}`);
    }
}
```
- constructor 是类的构造函数，用于初始化新创建的对象。
- 实例方法使用 this 关键字访问实例属性。

> kingscript 不支持使用静态变量

### 1.3 类的实例化
使用 new 关键字可以创建类的实例：
```kingscript
let person = new Person("Kingdee", 31);
person.greet(); // 输出: Hello, my name is Kindee
```

### 1.4 继承
KingScript 支持类的继承。使用 extends 关键字可以创建子类：
```kingscript
class Animal {
    name: string;

    constructor(name: string) {
        this.name = name;
    }

    move(): void {
        console.log(`${this.name} is moving`);
    }
}

class Dog extends Animal {
    bark(): void {
        console.log(`${this.name} is barking`);
    }
}

let dog = new Dog("Buddy");
dog.move();  // 输出: Buddy is moving
dog.bark(); // 输出: Buddy is barking

```

### 1.5 访问控制
KingScript 提供访问控制修饰符来限制类成员的可见性：
- public：默认，可以在任何地方访问。
- private：只能在声明它的类中访问。
- protected：可以在声明它的类及其子类中访问。
```kingscript
class Person {
    private name: string;

    protected sayHello(): void {
        console.log("Hello");
    }

    public getName(): string {
        return this.name;
    }
}

let person = new Person();
person.getName(); // 允许
person.sayHello(); // 错误，protected 方法不能在类外访问
```

### 1.6 类的常用场景
- 定义对象的行为和属性
- 实现继承和多态
- 提供代码复用性
- 方便管理复杂系统中的不同组件

通过类，我们可以更好地组织代码结构，提高程序的可维护性和扩展性。
